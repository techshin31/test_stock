# ⚠️ 리스크 관리 (Risk Management)

> [[주식 트레이딩 시스템|← 메인 지식 맵]]

## 핵심 원칙

> **좋은 분석보다 중요한 것은 틀렸을 때 손실을 얼마나 빨리 제한하는가이다**

트레이딩에서 리스크 관리는 선택이 아닌 **생존의 필수 조건**.

---

## FA 관점 리스크 관리

### 안전마진 (Margin of Safety)

- 내재 가치 대비 최소 **20~30% 이상 낮은 가격**에 진입
- 분석 오류에 대한 버퍼 확보
- → [[FA 체크리스트]] 참조

### 포트폴리오 분산

- 단일 종목 집중 위험 방지
- 섹터별·자산군별 분산
- → [[FA 개요]] 참조

### 이벤트 리스크

- 실적 발표, 가이던스 변경 등 불확실성 이벤트 전 포지션 조정
- → [[이벤트 분석]] 참조

## TA 관점 리스크 관리

### 1. 포지션 사이징 (1% 룰)

```python
계좌 리스크 = 전체 자산 × 1%
position_size = 계좌 리스크 / (진입가 - 손절가)
```

→ [[포지션 사이징]] 참조

### 2. 손절 (Stop-Loss) — 기계적 실행

```python
손절가 = 진입가 - (ATR × 1.5)
```

- **손절은 반드시 기계적으로 실행** — 예외 없음
- 손절 망설임 = 소손실 → 대손실 전환의 원인

### 3. 익절 (Take Profit)

```python
익절가 = 진입가 + (ATR × 3)   # Risk/Reward = 1:2
```

### 4. 트레일링 스탑 (Trailing Stop)

- 수익 발생 시 손절선 상향 조정
- 추세 지속 시 수익 극대화

→ [[청산 전략]] 참조

## 리스크 관리 3요소

| 요소 | 설명 |
|------|------|
| **포지션 사이징** | 한 포지션에 얼마를 투입할 것인가 |
| **손절 기준** | 언제 손실을 확정할 것인가 |
| **이벤트 리스크** | 불확실 이벤트 전 포지션 어떻게 관리할 것인가 |

## FA + TA 통합 리스크 관리 흐름

```
1. FA: 안전마진 확보된 종목만 선정
2. TA: 진입 타이밍 결정 (→ [[진입 전략]])
3. 포지션 사이징 계산 (→ [[포지션 사이징]])
4. 손절선/익절선 설정 (→ [[청산 전략]])
5. 이벤트 캘린더 확인 (→ [[이벤트 분석]])
6. 실시간 모니터링 및 트레일링 스탑 조정
```

## 계좌 관리 원칙

| 원칙 | 내용 |
|------|------|
| **1% 룰** | 1회 거래 최대 손실 = 계좌의 1% |
| **최대 동시 포지션** | 계좌 리스크 총합 5~10% 이내 |
| **드로우다운 한도** | 월 -10% 초과 시 포지션 축소 |
