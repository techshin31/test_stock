## 레이어별 세부 설계

### Layer 1: 종목 선정 (FA)

```python
# 유니버스 필터링
1. 매크로 환경 확인     → [[매크로 동태적 분석]]
2. 산업 라이프사이클    → [[탑다운 분석]]
3. 재무 스크리닝        → [[재무제표 분석]] + [[심화 메트릭]]
   - ROIC > WACC
   - FCF Yield > 5%
   - 부채비율 < 100%
4. 밸류에이션 확인      → [[정량적 분석]]
   - Forward PER < 업종 평균
   - PEG < 1.5
5. 내러티브 & 수급 확인 → [[내러티브]] + [[수급 분석]]
```

### Layer 2: 시장 상태 판단 (Regime)

```python
if ADX > 25 and MA_slope > 0:
    mode = "trend_following"     → [[MACD]] 추세 추종
elif ADX > 25 and MA_slope < 0:
    mode = "bear_trend"          → 관망
else:
    mode = "mean_reversion"      → [[Bollinger Bands]] + [[RSI]]
```

→ [[시장 상태 판단]] 참조

### Layer 3: 진입 신호 (Entry)

```python
# 추세 추종 모드
if ADX > 25 and MACD > Signal and 40 < RSI < 60 and price > MA20:
    if OBV 상승 추세:          # 수급 최종 확인
        생성_매수신호()

# 횡보 모드
if ADX < 25 and price <= BB_lower and RSI < 30:
    생성_매수신호()
```

→ [[진입 전략]] 참조

### Layer 4: 포지션 사이징

```python
ATR = 계산_ATR(14)
stop_loss = entry_price - (ATR * 1.5)
risk_amount = 계좌잔고 * 0.01        # 1% 룰
position_size = risk_amount / (entry_price - stop_loss)
```

→ [[포지션 사이징]] 참조

### Layer 5: 실행 (Execution)

```python
# 반드시 캔들 마감(Close) 기준
if 봉_마감:
    if 신호_유효():
        주문_실행(position_size)
        손절선_등록(stop_loss)
        익절선_등록(entry_price + ATR * 3)
```

→ [[멀티 타임프레임]] 참조

### Layer 6: 청산 (Exit)

```python
청산 조건 (우선순위 순):
1. 손절선 도달       → 즉시 청산
2. 트레일링 스탑     → 최고점 - ATR*1.5 하락 시
3. Dead Cross        → MACD 하향 돌파
4. 과매수            → RSI > 70 후 하락 전환
5. 익절선 도달       → 목표 수익 달성
6. 시간 기반         → N봉 초과 보유 시
```

→ [[청산 전략]] 참조

### Layer 7: 검증 (Backtesting)

```python
검증 절차:
1. Train/Test 분리 (7:3)
2. Walk-Forward 테스트
3. 다양한 종목 적용
4. 거래 비용 반영 (수수료 + 슬리피지)
5. 성과 지표 계산 (CAGR, MDD, 샤프 지수)
```

→ [[백테스트]] 참조
