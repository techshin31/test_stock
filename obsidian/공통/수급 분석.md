# 💰 수급 분석 (Flow of Funds)

> [[주식 트레이딩 시스템|← 메인 지식 맵]]

## 핵심 원칙

> **펀더멘털이 좋아도 돈이 안 들어오면 주가는 오르지 않는다**

수급은 FA와 TA를 연결하는 **실전 트레이딩의 핵심 변수**.

---

## 주요 수급 주체

| 주체 | 특징 | 시사점 |
|------|------|--------|
| **외국인** | 글로벌 매크로 관점, 대규모 자금 | 방향성 신뢰도 높음 |
| **기관** | 분기별 리밸런싱, 펀드 수급 | 중기 추세 형성 |
| **개인** | 단기 모멘텀 추종 경향 | 추세 말미 집중 |
| **ETF** | 패시브 자금, 지수 비중 변화 | 기계적·예측 가능 |

---

## 수급 체크 지표

### 외국인·기관 순매수/순매도

```
외국인 + 기관 동반 순매수 → 강력한 매수 신호
외국인 + 기관 동반 순매도 → 강력한 매도 신호
개인 순매수 + 기관 순매도 → 주의 (개인 물량 받기 구간 가능)
```

### ETF 자금 흐름

- ETF 자금 유입 → 구성 종목 기계적 매수
- ETF 자금 유출 → 구성 종목 기계적 매도
- **패시브 vs 액티브 자금** 비율 변화 모니터링

### 공매도 잔고

```
공매도 잔고 급증 → 하락 베팅 증가 (부정적 수급)
공매도 잔고 감소 + 주가 상승 → Short Squeeze 가능성 (급등 트리거)
```

---

## OBV로 수급 포착

기술적 지표로 수급의 흔적을 탐지.

```
가격 횡보 + OBV 상승 → 기관 매집 의심
가격 상승 + OBV 하락 → 거래량 없는 가짜 상승 (주의)
```

→ [[OBV]] 참조

---

## FA와의 연결

펀더멘털이 좋은 종목도 수급 유입이 없으면 주가 상승 불가.  
[[트레이딩 관점 FA]]에서 수급 확인이 체크리스트에 포함된 이유.

```
FA (내재 가치 확인) + 수급 (돈의 흐름 확인) = 매수 확신 상승
```

---

## 내러티브와의 관계

새로운 **내러티브 형성** → 수급 유입 → 주가 상승

```
스토리 형성 (AI, 전기차 등)
  → 외국인·기관 수급 유입
  → 밸류에이션 확장
  → 개인 참여 (추세 말미)
```

→ [[내러티브]] 참조

---

## 수급 이상 신호

| 신호 | 의미 |
|------|------|
| 주가 신고가 + 거래량 감소 | 추세 소멸 가능성 |
| 주가 하락 + 거래량 급증 | 투매(패닉 셀링) or 저점 매집 |
| 기관 지속 순매도 중 개인 순매수 | 고점 가능성 |
