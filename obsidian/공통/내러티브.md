# 🧠 내러티브 (Narrative)

> [[주식 트레이딩 시스템|← 메인 지식 맵]]

## 핵심 원칙

> **시장은 데이터보다 스토리(내러티브)에 먼저 반응한다**  
> 데이터를 해석하는 것이 아니라 **"이야기"를 먼저 가격에 반영**한다

---

## 내러티브 시장 사이클

```
1단계: 스토리 형성
       (AI 혁명, 전기차 전환, 바이오 혁신 등)
          ↓
2단계: 수급 유입
       (외국인·기관 선취매 → [[수급 분석]])
          ↓
3단계: 밸류에이션 확장 (Multiple Expansion)
       (PER 멀티플 상승 → [[트레이딩 관점 FA]])
          ↓
4단계: 분기점
       ├─ 실적 따라잡기 (내러티브 현실화) → 추가 상승
       └─ 실적 실망 (내러티브 붕괴) → 급락
```

---

## 내러티브 형성 요소

| 요소 | 예시 |
|------|------|
| **기술 혁신** | AI, 반도체, 자율주행 |
| **정책·규제** | 탄소중립 정책 → 신재생에너지 |
| **사회 변화** | 고령화 → 헬스케어·바이오 |
| **지정학** | 공급망 재편 → 방산·소재 |
| **거시경제** | 금리 하락 → 성장주 재평가 |

---

## 내러티브 변화 포착 방법

### 신호 1: 애널리스트 리포트 방향 전환

```
중립 → 매수 전환 급증
→ 기관 수급 유입 선행 신호
```

### 신호 2: 뉴스 흐름 및 검색 트렌드

- 관련 키워드 언급 빈도 급증
- 주요 언론 커버리지 확대

### 신호 3: 밸류에이션 멀티플 변화

```
PER 확장 없이 EPS만 증가 → 내러티브 없는 실적 장세
PER 급확장 + EPS 유지    → 순수 내러티브 장세 (버블 주의)
```

→ [[트레이딩 관점 FA]] 참조

---

## FA와 내러티브의 충돌

펀더멘털이 좋아도 **내러티브가 없으면** 주가 반응이 약할 수 있다.

```
FA 좋음 + 내러티브 없음 → 저평가 유지 (가치 함정 Value Trap)
FA 나쁨 + 내러티브 강함 → 과대평가 (버블 위험)
FA 좋음 + 내러티브 강함 → 최적 진입 기회
```

## 내러티브 붕괴 신호

| 신호 | 의미 |
|------|------|
| 실적 기대치 미달 반복 | 스토리 현실화 실패 |
| 경쟁사 등장 | 독점적 내러티브 훼손 |
| 규제 강화 | 성장 스토리 제한 |
| 기관 수급 이탈 | 내러티브 신뢰 상실 |

→ [[이벤트 분석]], [[수급 분석]] 참조

## 비즈니스 모델 전환과 내러티브

기업의 [[비즈니스 모델]] 전환이 새로운 내러티브를 만들어낸다.

```
예시: Netflix → 비디오 대여 → 스트리밍
     → "콘텐츠 플랫폼 지배" 내러티브 형성
     → 밸류에이션 급격히 재평가
```
