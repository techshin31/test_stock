# 🎯 진입 전략 (Entry Rules)

> [[TA 개요|← TA 개요]]

## 핵심 원칙

❌ 기존 문제: "고려" 수준의 모호한 조건  
✅ 개선: **수식 기반 명확한 조건** — 모든 조건 AND 충족 시에만 진입

---

## 전략 1: 추세 추종 (Trend Following)

[[시장 상태 판단]]에서 추세장 확인 후 사용

```python
매수 조건 (AND 조건):
  1. ADX > 25                    # 추세 확인 (필터)
  2. MACD > Signal               # 방향 결정 (최근 3봉 내 골든크로스)
  3. RSI: 40 ~ 60                # 과매수 아님 (진입 확인)
  4. 가격 > 20MA                 # 상승 추세 확인

→ 4가지 모두 충족 시 매수 진입
```

**보조 확인**: [[OBV]] 상승 추세 확인으로 수급 검증

---

## 전략 2: 횡보 역추세 (Mean Reversion)

[[시장 상태 판단]]에서 횡보장 확인 후 사용

```python
매수 조건 (AND 조건):
  1. ADX < 25                    # 횡보 확인 (필터)
  2. 가격 ≤ Bollinger 하단 밴드  # 눌림목 진입
  3. RSI < 30                    # 과매도 확인

→ 3가지 모두 충족 시 매수 진입 (반등 기대)
```

---

## 필터링 3단계 구조

```
Step 1: ADX    → 추세/횡보 판단 (전략 선택)
Step 2: MACD   → 진입 방향 결정
Step 3: RSI/OBV → 진입 확정 (타이밍)
```

→ [[TA 개요]] 참조

## 진입 주의사항

### 캔들 마감 기준 사용

```
실시간(Tick) 신호 → 봉 완성 전 신호 나타났다 사라질 수 있음
반드시 봉 마감(Close) 데이터 기준으로 진입 로직 실행
```

### 멀티 타임프레임 검증

```
일봉: 큰 추세 및 Regime 확인
15분/30분봉: 세부 진입 시점 결정
```

## 진입 후 즉시 설정

진입 직후 **반드시** 손절선과 익절선을 설정.

```
손절 = 진입가 - (ATR × 1.5)
익절 = 진입가 + (ATR × 3)
```

→ [[청산 전략]], [[포지션 사이징]], [[리스크 관리]] 참조

## 포지션 사이징 연동

```
계좌 리스크 = 전체 자산의 1%
position_size = (계좌 × 1%) / (진입가 - 손절가)
```
