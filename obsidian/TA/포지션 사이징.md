# 💰 포지션 사이징 (Position Sizing)

> [[TA 개요|← TA 개요]]

## 개념

> **계좌 생존에 가장 중요한 요소**  
> 얼마나 좋은 전략도 포지션 사이징이 잘못되면 파산(Ruin)에 이를 수 있다.

---

## Fixed Fractional 방식 (권장)

**계좌 자산의 일정 비율을 최대 리스크로 설정**하는 방식.

```python
# 기본 공식 (1% 룰)
계좌 리스크 = 전체 자산 × 1%

position_size = 계좌 리스크 / (entry_price - stop_loss_price)
```

### 예시

```
계좌: 10,000,000원
리스크 비율: 1% = 100,000원

진입가: 50,000원
손절가: 48,500원
손절 폭: 1,500원

position_size = 100,000 / 1,500 = 66주
```

---

## 1% 룰의 의의

- 1회 손절 시 계좌 손실 = 최대 1%
- 연속 10번 손절 시 계좌 손실 = 약 9.5% (복리 효과)
- **파산 없이 전략 개선 기회 유지**

---

## 리스크 비율 가이드

| 계좌 규모 / 경험 | 권장 리스크 비율 |
|----------------|----------------|
| 초보 / 검증 전 | 0.5% |
| 중급 / 검증 완료 | 1% |
| 고급 / 일관된 수익 | 1~2% |
| 최대 권장 | 2% |

> 2% 초과는 대부분의 경우 권장하지 않음

---

## ATR 기반 손절과 연동

```python
# ATR 기반 손절과 포지션 사이징 연동
stop_loss_distance = ATR × 1.5

position_size = (계좌 × risk_ratio) / stop_loss_distance
```

## 포지션 사이징 오류 유형

| 오류 | 결과 |
|------|------|
| 과대 포지션 | 1회 손절로 큰 손실, 심리적 판단 흔들림 |
| 과소 포지션 | 수익 기회 낭비 |
| 고정 금액 투입 | 손절 폭 무시, 일관성 없는 리스크 |
