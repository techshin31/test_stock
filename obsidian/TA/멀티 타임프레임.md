# 🕒 멀티 타임프레임 전략 (Multi-Timeframe)

> [[TA 개요|← TA 개요]]

## 개념

**상위 타임프레임으로 큰 흐름(방향)을 파악**하고,  
**하위 타임프레임으로 세부 진입 타이밍**을 잡는 전략.

> 단일 타임프레임 신호의 노이즈를 줄이고 신뢰도를 높인다.

---

## 기본 구조

```
상위 타임프레임 → 방향(Direction) + Regime 판단
하위 타임프레임 → 진입(Entry) 타이밍 최적화
```

---

## 권장 타임프레임 조합

### 주식 스윙 트레이딩

| 역할 | 타임프레임 | 확인할 것 |
|------|----------|----------|
| 큰 흐름 | **주봉** | 장기 추세 방향 |
| Regime 판단 | **일봉** | [[ADX]], 이동평균 기울기 |
| 진입 타이밍 | **15분/30분봉** | [[MACD]] 골든크로스, [[RSI]] |

### 데이트레이딩

| 역할 | 타임프레임 |
|------|----------|
| 방향 | 1시간봉 |
| 진입 | 5분/15분봉 |

---

## 구체적 활용 예시

```python
# 추세 추종 전략 (일봉 + 15분봉)

일봉 확인:
  ADX > 25               → 추세 확인
  MA 우상향              → 상승 추세 방향

15분봉 진입:
  MACD 골든크로스        → 매수 신호 발생 시 진입
  RSI: 40~60             → 과매수 아님 확인
```

---

## 타임프레임 불일치 처리

```
상위 TF: 상승 추세
하위 TF: 일시적 하락 (눌림목)
→ 하위 TF 하락을 역추세로 매수 기회로 활용 가능

상위 TF: 하락 추세
하위 TF: 반등 신호
→ 무시 or 매우 타이트한 손절로만 진입
```

| 중기 (1~3개월) | 일봉 |
| 장기 (6개월~) | 주봉, 월봉 |
