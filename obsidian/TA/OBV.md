# 📊 OBV (On-Balance Volume, 온밸런스볼륨)

> [[TA 개요|← TA 개요]] | 분류: **거래량 지표**

## 원리

**거래량이 가격에 선행한다**는 원리를 이용.  
가격 상승일의 거래량은 더하고, 하락일은 빼서 누적 계산.

```
종가 상승일: OBV = 전일 OBV + 거래량
종가 하락일: OBV = 전일 OBV - 거래량
종가 보합일: OBV = 전일 OBV
```

---

## 자동매매 활용

### 주요 신호

| 신호 | 조건 | 의미 |
|------|------|------|
| **매집 신호** | 가격 횡보 + OBV 상승 | 세력의 매집, 상승 가능성 |
| **약한 상승** | 가격 상승 + OBV 하락 | 거래량 없는 상승, 하락 전환 예고 |
| **추세 신뢰** | OBV 방향 = 가격 방향 | 현재 추세의 신뢰도 높음 |

### Divergence (다이버전스) — 선행 신호

```
강세 신호: 가격 하락/횡보 + OBV 상승 → 곧 상승 전환
약세 신호: 가격 상승        + OBV 하락 → 곧 하락 전환
```

## 주의사항

> OBV는 절대 수치보다 **방향과 기울기**가 중요  
> 주식 분할, 블록딜 같은 이벤트로 데이터 왜곡 가능

## 수급 분석과의 관계

OBV는 **기관·외국인 수급**의 기술적 흔적을 포착하는 도구.

```
OBV 지속 상승 + 주가 박스권  → 기관 매집 가능성
OBV 지속 하락 + 주가 횡보    → 기관 분산 매도 가능성
```
