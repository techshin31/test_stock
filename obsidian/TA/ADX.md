# 📊 ADX (Average Directional Index, 평균방향지수)

> [[TA 개요|← TA 개요]] | 분류: **추세 강도 지표 (필터)**

## 원리

현재 시장이 한 방향으로 **얼마나 강하게 움직이는지** 0~100으로 수치화.  
⚠️ 방향(상승/하락)은 알려주지 않음 — 강도만 측정.

```
ADX = +DI와 -DI 차이의 이동평균
+DI: 상승 방향 강도
-DI: 하락 방향 강도
```

---

## 자동매매 활용

### ADX 값 해석

| ADX 값 | 시장 상태 | 전략 선택 |
|--------|-----------|----------|
| **ADX < 25** | 추세 없음 (횡보) | [[Bollinger Bands]], [[RSI]] 횡보 전략 |
| **ADX 25~40** | 추세 형성 | [[MACD]] 추세 추종 신뢰도 상승 |
| **ADX > 40** | 강한 추세 | 추세 추종 전략 유효 |

### 방향 판단: +DI / -DI

```
+DI > -DI → 상승 추세
-DI > +DI → 하락 추세

+DI와 -DI 크로스 → 추세 전환 신호
```

---

## 주의사항

> ADX는 **방향을 알려주지 않는다**  
> 급락장에서도 ADX는 높게 나타날 수 있음

**필수 조합**: +DI/-DI 또는 이동평균으로 방향 확인

```python
if ADX > 25:
    if DI_plus > DI_minus:
        상승 추세 → 매수 전략
    elif DI_minus > DI_plus:
        하락 추세 → 공매도 or 관망
```

---

## 시장 상태 판단에서의 역할

ADX는 **전략 선택 필터**로서 가장 중요한 지표.

```python
# 권장 Regime Detection 로직
if ADX > 25 and MA_slope > 0:
    → 상승 추세 → MACD 기반 추세 추종
elif ADX > 25 and MA_slope < 0:
    → 하락 추세 → 관망 or 역추세 주의
else:
    → 횡보 → Bollinger Bands / RSI 활용
```

## 진입 전략에서의 활용

모든 진입 전략의 **1차 필터**로 사용:

```
Step 1: ADX로 추세/횡보 판단   ← 여기서 전략 갈림
Step 2: MACD로 방향 결정
Step 3: RSI/OBV로 진입 확정
```
