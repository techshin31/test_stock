# ⛔ 청산 전략 (Exit Rules)

> [[TA 개요|← TA 개요]]

## 핵심 원칙

> 자동매매에서 **'잘 사는 것'보다 '잘 파는 것'**이 더 중요하다.

좋은 진입보다 **일관된 청산 규칙**이 계좌 생존의 핵심.

---

## 5가지 청산 방법

### 1. 손절 (Stop-Loss) — ATR 기반

```python
손절 = entry_price - (ATR × 1.5)
```

- **ATR**: 최근 N일 평균 변동폭 → 시장 변동성에 비례한 손절선
- 고정 % 손절보다 ATR 기반이 더 합리적

### 2. 익절 (Take Profit) — ATR 기반

```python
익절 = entry_price + (ATR × 3)
```

- **Risk/Reward Ratio = 1:2** (손절 1.5 ATR : 익절 3 ATR)
- 기대값이 양수가 되는 기본 비율

### 3. 트레일링 스탑 (Trailing Stop)

```python
# 수익 발생 시 손절선을 상향 조정
현재 가격이 최고점 갱신 시:
    손절선 = 현재 최고점 - (ATR × 1.5)
```

- 추세가 계속되는 동안 수익 극대화
- 추세 반전 시 자동 청산

### 4. 신호 기반 청산 (Signal-Based)

```python
청산 조건:
  - MACD Dead Cross 발생
  - RSI > 70 후 하락 전환
  - ADX 급격한 하락 (추세 소멸)
```

→ [[MACD]], [[RSI]], [[ADX]] 참조

### 5. 시간 기반 청산 (Time-Based)

```python
# N봉 이상 보유 시 강제 청산 (기회비용 방지)
if holding_candles >= N:
    강제 청산
```

---

## 청산 우선순위

```
1순위: 손절 (Stop-Loss)       ← 반드시 먼저 실행
2순위: 트레일링 스탑
3순위: 신호 기반 청산
4순위: 익절 (Take Profit)
5순위: 시간 기반 청산
```

## 심리적 함정 주의

| 함정 | 설명 | 대응 |
|------|------|------|
| **손절 망설임** | "곧 반등할 것" 심리 | 기계적 손절 규칙 고수 |
| **익절 조기 청산** | 수익 실현 조급증 | 트레일링 스탑 활용 |
| **평균 단가 낮추기** | 하락 종목 추가 매수 | 절대 금지 |
