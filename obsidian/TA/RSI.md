# 📊 RSI (상대강도지수, Relative Strength Index)

> [[TA 개요|← TA 개요]] | 분류: **모멘텀 지표**

## 원리

최근 N일(기본 14일) 동안의 **상승폭과 하락폭의 비율**을 0~100으로 수치화.  
에너지(모멘텀)의 강도를 측정.

```
RS = 평균 상승폭 / 평균 하락폭
RSI = 100 - (100 / (1 + RS))
```

---

## 자동매매 활용

### 기본 신호

| 구간 | 의미 | 전략 |
|------|------|------|
| **RSI < 30** | 과매도 | 매수 고려 |
| **RSI > 70** | 과매수 | 매도 고려 |
| **RSI 40~60** | 중립 | 추세 추종 진입 확인용 |

### Divergence (다이버전스) — 강력한 반전 신호

```
강세 다이버전스: 주가 신저점 갱신 + RSI는 저점 상승 → 반등 신호
약세 다이버전스: 주가 신고점 갱신 + RSI는 고점 하락 → 하락 전환 신호
```

---

## 주의사항

> RSI가 30에 닿았다고 **무조건 반등하지 않는다**  
> 하락 추세가 강하면 20대에서도 계속 하락 가능

**필수 조합**: [[ADX]]로 추세/횡보 확인 후 사용

```
횡보장 (ADX < 25) → RSI 신뢰도 높음
추세장 (ADX > 25) → RSI 단독 사용 위험
```

## 시장 상태별 활용

| 시장 상태 | 활용 방법 |
|-----------|----------|
| **횡보장** | 30/70 구간에서 역추세 매매 |
| **추세장** | 40~60 유지 = 추세 건전성 확인 지표 |

## 진입 조건 예시

```python
# 횡보 역추세 전략
매수 조건:
  ADX < 25          # 횡보 확인
  RSI < 30          # 과매도
  Bollinger 하단 터치  # 추가 확인

# 추세 추종 전략 (확인용)
매수 조건:
  ADX > 25          # 추세 확인
  MACD 골든크로스   # 방향 결정
  RSI: 40~60        # 과매수 아님 확인
```
