# 📊 MACD (Moving Average Convergence Divergence)

> [[TA 개요|← TA 개요]] | 분류: **추세 + 모멘텀 지표**

## 원리

단기(12일)와 장기(26일) **지수이동평균(EMA)의 차이**를 이용하여  
추세의 전환점과 강도를 포착.

```
MACD선   = EMA(12) - EMA(26)
시그널선  = EMA(MACD선, 9)   ← MACD선의 9일 EMA
히스토그램 = MACD선 - 시그널선
```

---

## 자동매매 활용

### 기본 신호

| 신호 | 조건 | 의미 |
|------|------|------|
| **Golden Cross** | MACD선 > 시그널선 상향 돌파 | 매수 신호 |
| **Dead Cross** | MACD선 < 시그널선 하향 돌파 | 매도 신호 |
| **0선 상향 돌파** | MACD선이 0 이상 | 상승 모멘텀 강화 |
| **0선 하향 돌파** | MACD선이 0 이하 | 하락 모멘텀 강화 |

### 히스토그램 활용

```
히스토그램 증가 → 모멘텀 강화
히스토그램 감소 → 모멘텀 약화 (추세 전환 예고)
```

---

## 주의사항

> **대표적인 후행 지표**  
> 신호 발생 시 이미 가격의 상당 부분이 진행된 경우 多

```
변동성 심한 종목 → Whipsaw (허위 신호) 多 발생
대응: ADX로 추세 강도 확인 후 사용
```

## 시장 상태별 활용

| 시장 상태 | MACD 신뢰도 |
|-----------|------------|
| **추세장** (ADX > 25) | 높음 — 추세 추종 전략 핵심 |
| **횡보장** (ADX < 25) | 낮음 — Whipsaw 위험 |

## 진입 조건 예시 (추세 추종 전략)

```python
매수 조건:
  ADX > 25                      # 추세 확인
  MACD > Signal (최근 3봉 내 골든크로스)
  RSI: 40~60                    # 과매수 아님
  가격 > 20MA                   # 상승 추세 확인

→ 모든 조건 충족 시 진입
```
