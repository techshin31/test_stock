# 📈 TA 개요 — Technical Analysis

> [[주식 트레이딩 시스템|← 메인 지식 맵]]

## 개념

**차트와 지표**를 이용해 진입(Entry)·청산(Exit) 타이밍을 결정하는 분석 체계.  
자동매매 알고리즘의 로직 설계에 직접 적용.

---

## 5대 핵심 지표

| 지표 | 분류 | 주요 용도 | 잘 작동하는 시장 |
|------|------|-----------|----------------|
| [[Bollinger Bands]] | 변동성 | 눌림목·돌파 포착 | 횡보장 |
| [[RSI]] | 모멘텀 | 과매수·과매도 감지 | 횡보장 |
| [[MACD]] | 추세+모멘텀 | 추세 전환 및 강도 확인 | 추세장 |
| [[ADX]] | 강도 | 전략 선택을 위한 필터링 | 모든 시장 |
| [[OBV]] | 거래량 | 가격 움직임의 선행 확인 | 모든 시장 |

---

## 시스템 설계 3대 원칙

### 1. 지표의 조합과 필터링 (Multi-Indicator)

단일 지표는 속임수가 많다 → 단계적 필터링 필수

```
Step 1: ADX  →  추세 유무 판단 (Filter)
Step 2: MACD →  진입 방향 결정 (Direction)
Step 3: RSI / OBV  →  진입 확정 (Confirmation)
```

### 2. 타임프레임 및 데이터 확정

- **멀티 타임프레임**: 일봉으로 큰 추세, 15분봉으로 세부 진입
- **캔들 마감 기준**: 봉 완성 전 신호는 허위 신호 → [[멀티 타임프레임]] 참조

### 3. 리스크 관리 로직 (Exit Strategy)

'잘 사는 것'보다 **'잘 파는 것'**이 더 중요하다.
- Stop-loss: 기계적 손절
- Trailing Stop: 수익 보존형 손절선
- Position Sizing: 자산 대비 투입 비중 조절
→ [[청산 전략]], [[포지션 사이징]], [[리스크 관리]] 참조

## 전략 선택 로직

```
ADX > 25 → 추세장 전략 (MACD, 이동평균 추세 추종)
ADX < 25 → 횡보장 전략 (Bollinger Bands, RSI 역추세)
```
