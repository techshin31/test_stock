## 🔄 FA + TA 통합 전략 흐름

```
종목 선정 (FA)          타이밍 결정 (TA)
      │                        │
  내재 가치 파악     →    진입 시점 포착
  펀더멘털 검증      →    기술 지표 확인
  리스크 평가        →    손절·익절 설정
```

> 📌 **핵심 원칙**: 펀더멘털로 *무엇을 살지* → 기술적 분석으로 *언제 살지* 결정
