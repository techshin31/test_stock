# 📋 재무제표 분석

> [[FA 개요|← FA 개요]]

## 세 가지 핵심 재무보고서

| 보고서 | 주요 내용 | 핵심 질문 |
|--------|----------|----------|
| **손익계산서** | 수익, 비용, 순이익 | "얼마나 벌었는가?" |
| **대차대조표** | 자산, 부채, 자본 | "재무 구조는 건전한가?" |
| **현금흐름표** | 영업·투자·재무 현금 흐름 | "실제 현금은 얼마인가?" |

---

## 손익계산서 (Income Statement)

```
매출액 (Revenue)
  - 매출원가 (COGS)
= 매출총이익 (Gross Profit)
  - 판관비 (SG&A)
= 영업이익 (Operating Income / EBIT)
  - 이자비용
  - 세금
= 순이익 (Net Income)
```

### 트레이딩 포인트
- **이익의 질** 확인 → [[이벤트 분석]] 참조
- 일회성 이익 vs 본업 경쟁력 구분

---

## 대차대조표 (Balance Sheet)

```
자산 (Assets)  =  부채 (Liabilities) + 자본 (Equity)

유동자산: 현금, 재고, 매출채권
비유동자산: 토지, 설비, 무형자산

유동부채: 단기차입금, 매입채무
비유동부채: 장기부채
```

### 트레이딩 포인트
- 부채비율: 재무 레버리지 리스크 확인
- 유동비율: 단기 지급 능력

---

## 현금흐름표 (Cash Flow Statement)

```
영업활동 현금흐름 (Operating CF)    ← 핵심 지표
  투자활동 현금흐름 (Investing CF)
  재무활동 현금흐름 (Financing CF)
= 기말 현금 잔액
```

### FCF (잉여현금흐름)
- **FCF = 영업 CF - 설비투자(CAPEX)**
- 장부상 이익보다 조작이 어려워 신뢰도 높음
- → [[심화 메트릭]] 참조

---

## 회계 기준

- **IFRS**: 한국 포함 대부분 국가 채택 (국제 비교 가능)
- **GAAP**: 미국 기준 (미국 상장사)

## [[정량적 분석]]에서 활용하는 핵심 비율

| 비율 | 계산식 | 의미 |
|------|--------|------|
| PER | 주가 / EPS | 수익 대비 주가 수준 |
| ROE | 순이익 / 자기자본 | 자본 활용 수익성 |
| EPS | 순이익 / 발행주식수 | 주당 순이익 |
| 부채비율 | 총부채 / 자기자본 | 재무 레버리지 |
