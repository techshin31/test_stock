# ✅ FA 체크리스트 (Fundamental Analysis Checklist)

> [[FA 개요|← FA 개요]]

## 기본 펀더멘털 체크리스트

### 1단계: 거시환경 확인 (→ [[매크로 동태적 분석]])
- [ ] 금리 방향성 파악 (상승/하락/동결)
- [ ] 환율 영향 분석 (수출·수입 기업 구분)
- [ ] 지정학적 리스크 및 공급망 점검

### 2단계: 산업 분석 (→ [[탑다운 분석]])
- [ ] 산업 라이프사이클 단계 파악
- [ ] [[경쟁적 위치|포터의 5가지 힘]] 분석
- [ ] 규제 환경 및 정책 리스크 확인

### 3단계: 기업 재무 분석 (→ [[재무제표 분석]])
- [ ] 손익계산서: 매출 성장률·이익률 추이
- [ ] 대차대조표: 부채비율·유동비율 확인
- [ ] 현금흐름표: FCF 창출 능력 확인 (→ [[심화 메트릭]])

## 트레이딩 분석가 핵심 체크리스트

> 리포트 작성·전략 수립 시 최종 검토 사항 (→ [[트레이딩 관점 FA]])

### 1. 컨센서스 대비 상회 여부
- [ ] 시장 기대치(Consensus) 대비 실제 EPS 확인
- [ ] Earnings Surprise 방향 (+/-) 파악
- [ ] 애널리스트 Revision 방향 확인

### 2. 이익의 질 (Quality of Earnings)
- [ ] 일회성 이익(자산 매각, 세제 혜택) 제거 후 확인
- [ ] 본업 경쟁력 강화에 의한 이익 개선인가?
- [ ] FCF와 순이익의 괴리 여부 (→ [[심화 메트릭]])

### 3. 내러티브의 변화 (→ [[내러티브]])
- [ ] 시장이 해당 산업을 바라보는 관점이 바뀌고 있는가?
- [ ] 새로운 스토리(AI, 전기차 등)가 형성되고 있는가?
- [ ] 기존 내러티브 붕괴 신호가 있는가?

### 4. 안전마진 (Margin of Safety)
- [ ] 계산된 내재 가치 대비 현재 주가 확인
- [ ] 최소 **20~30% 이상의 괴리율** 확보 여부

## 이벤트 기반 체크리스트 (→ [[이벤트 분석]])

- [ ] 가이던스(Guidance) 발표 일정 확인
- [ ] 자사주 매입·소각 계획 파악
- [ ] 지배구조 이슈 (코리아 디스카운트 요인) 점검
- [ ] M&A·분할·합병 관련 공시 확인

---

## 수급 체크리스트 (→ [[수급 분석]])

- [ ] 외국인 순매수/순매도 방향
- [ ] 기관 수급 확인
- [ ] ETF 자금 유입/유출 방향
- [ ] 공매도 잔고 및 변화 추이

## 리스크 관리 체크리스트 (→ [[리스크 관리]])

- [ ] 포지션 사이징 결정
- [ ] 손절 기준(Stop Loss) 설정
- [ ] 이벤트 리스크 대응 계획 수립
