# 📊 심화 메트릭 (Advanced Metrics)

> [[FA 개요|← FA 개요]]

## 개요

단순 재무제표를 넘어 **시장의 기대치와 실제 성과의 괴리**를 파악하는 고급 지표들.  
트레이딩 분석가가 반드시 숙지해야 할 핵심 메트릭.

---

## 1. 수익성·효율성 심화 지표

### ROIC (투하자본수익률)

```
ROIC = 세후영업이익 (NOPAT) / 투하자본 (Invested Capital)
```

- 기업이 실제 영업에 투입한 자본 대비 수익 효율 측정
- **ROE의 한계 보완**: 부채 레버리지 효과를 제거한 순수 영업 효율
- ROIC > WACC(가중평균자본비용) → **가치 창출 기업**

### FCF (잉여현금흐름, Free Cash Flow)

```
FCF = 영업활동 현금흐름 - CAPEX (설비투자)
```

- 실제 배당·재투자·부채 상환에 쓸 수 있는 **'진짜 현금'**
- 장부상 이익보다 **조작이 어려워** 신뢰도 높음
- → [[재무제표 분석]] 참조

### 가동률 & 재고회전율

- 제조업 기반 기업의 **업황 정점(Peak-out)** 포착에 필수
- 선행 지표 성격: 재고 증가 → 수요 둔화 신호

---

## 2. 밸류에이션 다각화

### PEG 배수 (Price/Earnings to Growth)

```
PEG = PER / EPS 성장률 (%)
```

| PEG 값 | 해석 |
|--------|------|
| PEG < 1 | 저평가 가능성 (성장 대비 저렴) |
| PEG = 1 | 적정 가치 |
| PEG > 1 | 고평가 가능성 |

- 성장이 빠른 기업이 **고PER인지 적정 가치인지** 판단할 때 유용
- → [[정량적 분석]] 참조

### EV/EBITDA

```
EV (기업가치) = 시가총액 + 순부채
EV/EBITDA = EV / EBITDA
```

- **현금창출능력 대비 기업가치** 비교
- 감가상각비가 큰 **장치 산업** 분석 시 PER보다 정확
- 부채 구조가 다른 기업 간 비교 시 유용

---

## 3. 업황 사이클 파악 지표

| 지표 | 포착하는 것 |
|------|------------|
| 재고회전율 하락 | 수요 둔화 선행 신호 |
| 가동률 하락 | 공급 과잉·업황 하락 |
| FCF 감소 + 이익 증가 | 이익의 질 저하 (위험 신호) |
