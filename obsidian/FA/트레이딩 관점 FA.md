# ⚡ 트레이딩 관점 FA (Trading-Oriented Fundamental)

> [[FA 개요|← FA 개요]]

## 개요

기존 FA를 **실전 트레이딩 환경**에 맞게 확장한 실무 중심 프레임워크.  
펀더멘털은 "정답"이 아니라 **시장 기대와 현실의 차이를 이용하는 도구**다.

---

## 1. 시간 프레임(Time Horizon) 구분

펀더멘털은 **시간에 따라 다르게 해석**해야 한다.

| 구분 | 핵심 포인트 |
|------|------------|
| **단기** (1일 ~ 2주) | 이벤트, 뉴스, [[수급 분석\|수급]] |
| **중기** (1~3개월) | 실적, 가이던스 변화 ([[이벤트 분석]]) |
| **장기** (6개월~) | 산업 구조, [[경쟁적 위치\|경제적 해자]] |

> 같은 데이터라도 시간 축에 따라 해석이 달라진다

---

## 2. 컨센서스 vs 실제 (Earnings Surprise)

주가는 실적 자체보다 **시장 기대 대비 차이**에 반응한다.

```
주가 반응 = f(실제 실적 - 시장 기대치)
```

### 핵심 요소
- **Earnings Surprise**: 실제 vs 예상 EPS 차이
- **Revision**: 추정치 상향/하향 방향
- **Pre-earnings positioning**: 사전 기대감 형성 여부

> 📌 좋은 실적 ≠ 상승  
> 📌 **"예상보다 좋은 실적"**만 상승 요인

## 4. 내러티브(Narrative) 중심 시장 구조

시장은 데이터보다 **스토리**에 먼저 반응한다.

→ [[내러티브]] 참조

### 시장 사이클
1. 스토리 형성 (AI, 전기차 등)
2. 자금 유입
3. 밸류에이션 확장
4. 실적 따라잡기 또는 붕괴

---

## 5. 밸류에이션 멀티플 변화

주가는 단순 이익이 아니라 **멀티플 변화**에도 크게 영향받는다.

```
주가 상승 = 이익(EPS) 증가 + PER 멀티플 상승
```

| 개념 | 원인 |
|------|------|
| **Multiple Expansion** (멀티플 확장) | 금리 하락, 유동성 확대 |
| **Multiple Compression** (멀티플 축소) | 금리 상승, 리스크 오프 |

→ [[매크로 동태적 분석]] 참조

---

## 6. 매크로 → 마이크로 연결

트레이딩은 지표를 나열하는 것이 아니라  
**인과관계로 연결하는 것**이 핵심이다.

```
금리 상승 → 할인율 상승 → 성장주 밸류에이션 하락 → 기술주 약세
```

---

## 7. 팩터(Factor) 기반 분석

FA와 퀀트를 연결하는 핵심 개념.

| 팩터 | 설명 |
|------|------|
| **Value** | 저평가 종목 선별 (PER, PBR) |
| **Growth** | 성장률 높은 종목 (EPS 성장) |
| **Quality** | 수익성·안정성 높은 기업 (ROIC, FCF) |
| **Momentum** | 상승 추세 지속 종목 (→ [[MACD]], [[ADX]]) |
