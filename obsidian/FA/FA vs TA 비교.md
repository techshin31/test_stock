# ⚖️ FA vs TA 비교

> [[주식 트레이딩 시스템|← 메인 지식 맵]]

## 핵심 차이점

| 구분 | [[FA 개요\|펀더멘털 분석 (FA)]] | [[TA 개요\|기술적 분석 (TA)]] |
|------|------------------------------|------------------------------|
| 데이터 소스 | 기업 보고서, 재무제표 | 차트, 거래 데이터 |
| 적용 대상 | 상장·비상장 기업 모두 | 거래 이력 있는 금융상품 |
| 신뢰 기반 | 기업의 실제 성과 | 시장 참여자의 행동 패턴 |
| 주요 활용 | 올바른 자산 선택, 공정 가격 범위 | 진입·청산 시점 파악 |
| 시간 지평 | 중장기 (6개월~) | 단기~중기 (당일~3개월) |

---

## 통합 전략

> 두 방법은 상호 보완적: **FA로 종목 선정 → TA로 타이밍 결정**

```
FA (무엇을?)              TA (언제?)
    │                         │
내재 가치 계산      →    진입 시점 포착
저평가 종목 선정    →    기술 지표 확인
리스크 평가         →    손절선 설정
```

---

## FA가 강점인 영역

- 우량 기업 장기 보유 종목 발굴
- 비상장 기업·IPO 전 가치 평가
- 경제적 해자([[정성적 분석]] 참조) 보유 기업 선별

## TA가 강점인 영역

- 진입·청산 타이밍 최적화 (→ [[진입 전략]], [[청산 전략]])
- 단기 수급 흐름 포착 (→ [[수급 분석]])
- 추세 강도 파악 (→ [[ADX]], [[MACD]])

---

## 실전 활용 예시

1. FA로 반도체 섹터 유망 기업 선별
2. [[탑다운 분석]]으로 거시환경 확인
3. [[시장 상태 판단]]으로 추세/횡보 파악
4. [[진입 전략]]으로 매수 타이밍 결정
5. [[리스크 관리]]로 손절·익절 설정
