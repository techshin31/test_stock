# 📅 이벤트 분석 (Event-Driven Fundamental)

> [[FA 개요|← FA 개요]]

## 개념

재무제표는 '과거'의 기록.  
트레이더는 **'미래'를 바꿀 이벤트**를 선제적으로 체크해야 한다.

---

## 1. 가이던스 (Guidance) 분석

> 확정 실적보다 경영진이 발표하는 **다음 분기/연도 전망치**가 주가 방향성에 더 큰 영향

### 핵심 체크포인트
- **가이던스 상향(Raise)**: 강력한 매수 시그널
- **가이던스 하향(Cut)**: 강력한 매도 시그널  
- **가이던스 유지**: 시장 기대 대비 상회/하회 여부 판단

### Earnings Surprise 메커니즘
```
실제 EPS > 컨센서스 EPS  →  Positive Surprise  →  주가 상승
실제 EPS < 컨센서스 EPS  →  Negative Surprise  →  주가 하락
```

## 2. 주주 환원 정책

기업의 **자본 효율성 개선** 의지를 나타내는 강력한 펀더멘털 개선 신호.

| 정책 | 의미 |
|------|------|
| **자사주 매입** | 주당 가치 상승 (EPS 개선) |
| **자사주 소각** | 영구적 주식수 감소 → 주주가치 극대화 |
| **배당 성향 증가** | 잉여현금흐름([[심화 메트릭]]) 주주 환원 의지 |
| **배당 성향 감소** | 성장 투자 우선 or 현금흐름 악화 신호 |

## 3. 지배구조 (Governance)

한국 시장 특유의 **코리아 디스카운트** 주요 요인.

| 이슈 | 설명 |
|------|------|
| **계열사 일감 몰아주기** | 소수주주 이익 침해 |
| **인적 분할** | 지주회사 전환 시 주주 가치 희석 우려 |
| **이사회 독립성** | 오너 중심 의사결정 → 소수주주 보호 미흡 |
| **배당 불투명성** | 예측 불가능한 배당 정책 |

→ [[정성적 분석]] 참조

## 4. 주요 이벤트 캘린더

| 이벤트 | 주가 영향 시점 |
|--------|--------------|
| 실적 발표 (Earnings) | 발표 전후 |
| 가이던스 수정 | 즉각 반응 |
| 자사주 매입 발표 | 발표 당일 |
| 주주총회 | 시즌 전후 |
| 공정공시 (IR) | 발표 즉시 |
| M&A 발표 | 발표 당일 (피인수 기업 급등) |
