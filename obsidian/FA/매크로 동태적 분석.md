# 🌐 매크로 동태적 분석 (Dynamic Macro)

> [[탑다운 분석|← 탑다운 분석]]

## 개념

단순히 지표를 나열하는 것이 아니라 **지표 간의 인과관계**를 파악하는 동태적 접근.  
거시 환경 변화가 개별 기업 밸류에이션에 미치는 경로를 이해해야 함.

---

## 1. 금리와 밸류에이션의 상관관계

```
금리 상승
  → 할인율(Discount Rate) 상승
  → 미래 현금흐름의 현재가치 하락
  → 성장주(High-Growth Tech) 밸류에이션 하락
  → PER(멀티플) 수축 → [[트레이딩 관점 FA]] 참조
```

```
금리 하락 (유동성 확대)
  → 할인율 하락
  → 미래 가치 현재가치 상승
  → 성장주 멀티플 확장
```

### 실전 적용
- 금리 인상기 → 가치주·배당주 비중 확대
- 금리 인하기 → 성장주·기술주 비중 확대

---

## 2. 환율 민감도 분석

| 기업 유형 | 원화 약세 시 | 원화 강세 시 |
|-----------|------------|------------|
| **수출 비중 高** (자동차, 반도체) | 수익 증가 (호재) | 수익 감소 (악재) |
| **원자재 수입 高** (음식, 에너지) | 원가 상승 (악재) | 원가 감소 (호재) |
| **외화 부채 多** | 환손실 위험 | 환이익 발생 |

---

## 3. 지정학적 리스크 & 공급망 (SCM)

- 특정 국가 의존도가 높은 부품·소재의 경우 **펀더멘털 외적 변수**로 급변 가능
- 반도체: 미국-중국 분쟁 영향
- 배터리: 리튬·코발트 공급망 집중 리스크
- **공급망 다변화 여부** → 기업 리스크 판단 기준

---

## 4. 매크로 → 마이크로 연결 구조 예시

```
원자재 가격 상승
  → 제조업 원가 상승
  → 마진 압박
  → 실적 하향 가능성
  → 컨센서스 하회 우려 (→ [[FA 체크리스트]])
```

```
중국 경기 부양
  → 수출 수요 회복 기대
  → 한국 수출 기업 실적 개선 기대
  → 내러티브 형성 (→ [[내러티브]])
  → 기관 수급 유입 (→ [[수급 분석]])
```

---

## 5. 주요 매크로 지표

| 지표 | 주식시장 시사점 |
|------|----------------|
| GDP 성장률 | 경기 사이클 판단 |
| 인플레이션 (CPI) | 금리 방향성 예측 |
| 실업률 | 소비 동향, 경기 과열/냉각 |
| PMI | 제조업 경기 선행 지표 |
| 정부부채/GDP | 재정 리스크 |
