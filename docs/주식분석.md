# 1. 금융 기초 및 시장 도메인 (The Logic)
> 가장 먼저 공부해야 할 분야입니다. 무엇을 분석할지 모른다면 아무리 좋은 알고리즘도 소용이 없기 때문이죠.

- 기본적 분석: 재무제표 읽는 법(손익계산서, 대차대조표), 주요 지표(PER, PBR, ROE)의 의미를 파악해야 합니다.

- 기술적 분석: 캔들 차트, 이동평균선, RSI, MACD 같은 보조지표의 계산 원리를 알아야 로직으로 구현할 수 있습니다.

- 시장 구조: 호가창의 원리, 체결 엔진, 매매 체결 단위 등 시장이 실제로 어떻게 돌아가는지 이해해야 합니다.

# 2. 데이터 엔지니어링 (The Fuel)
> 주식 데이터는 양이 방대하고 실시간성이 중요합니다. 데이터를 어떻게 가져오고 저장할지가 시스템의 성능을 결정합니다.

- API 활용: 증권사 API(Open API), FinanceData, Yahoo Finance 등에서 데이터를 당겨오는 법을 익히세요.

- 시계열 데이터베이스: 주가 데이터는 시간에 따라 변하는 '시계열 데이터'입니다. 이를 효율적으로 처리하기 위한 DB(Pandas, InfluxDB, 혹은 가벼운 SQLite) 활용법이 필수입니다.

# 3. 퀀트 및 데이터 과학 (The Brain)
> 수집한 데이터를 바탕으로 실제 '전략'을 짜는 단계입니다.

- 통계학 기초: 평균 회귀, 상관계수 등 기초 통계 지식이 있어야 전략의 유효성을 검증할 수 있습니다.

- 백테스팅(Backtesting): 내가 만든 전략을 과거 데이터에 대입했을 때 수익이 났는지 확인하는 로직을 직접 짜보거나 라이브러리(Backtrader 등)를 공부해야 합니다.
