# WICS 대분류 기반 DART 분석 초안

## 1. 목적

WICS 대분류를 기준으로 기업을 묶고, DART에서 수집한 재무제표 및 사업보고서 정보를 결합하여 섹터별 특성과 섹터 내 기업의 상대적 위치를 분석한다.

이 분석의 1차 목표는 다음과 같다.

- WICS 대분류별 재무 특성 파악
- 동일 대분류 내 기업 상대비교 기준 마련
- 향후 리포트, 추천, 요약 기능에 재사용 가능한 분석 레이어 구축

## 2. 핵심 질문

이번 분석은 아래 질문에 답할 수 있도록 설계한다.

- 각 WICS 대분류는 재무적으로 어떤 특징을 가지는가?
- 같은 대분류 안에서 어떤 기업이 성장성, 수익성, 안정성이 높은가?
- 특정 기업은 자기 섹터 평균 또는 중앙값 대비 어느 위치에 있는가?
- 사업보고서 텍스트는 재무 특성을 어떻게 설명해주는가?

## 3. 분석 범위

### 3.1 1차 범위

우선은 숫자 기반 분석에 집중한다.

- WICS 대분류 매핑
- DART 재무제표 기반 핵심 지표 계산
- 섹터별 기준선 산출
- 기업별 섹터 내 상대평가

### 3.2 2차 범위

1차 결과가 안정화된 뒤 텍스트 분석을 확장한다.

- 사업보고서 사업개요 텍스트 수집
- 사업부문, 원재료, 수출/내수, 투자계획 관련 내용 요약
- 텍스트와 재무지표의 연결 해석

## 4. 데이터 소스

현재 활용 가능한 주요 데이터는 다음과 같다.

- `etl/wics/data/csv/wics_company_2026.csv`
- `etl/company/data/dart_company_2026.csv`
- `etl/company/data/balance_sheet_2021.csv` ~ `balance_sheet_2025.csv`
- `etl/company/data/income_statement_2021.csv` ~ `income_statement_2025.csv`
- `etl/company/data/cash_flow_2021.csv` ~ `cash_flow_2025.csv`

필요 시 사업보고서 본문 텍스트는 `etl/company/dart_api.py`의 수집 로직을 확장하여 활용한다.

## 5. 분석 레이어 설계

분석은 3개 레이어로 분리한다.

### 5.1 기초 레이어

기업-연도 단위로 원천 데이터를 정리한다.

주요 컬럼 예시:

- 회사명
- 종목코드
- DART 코드
- 기준연도
- WICS 대분류
- 재무제표 원천 계정값

### 5.2 지표 레이어

원천 재무제표를 기반으로 핵심 재무지표를 계산한다.

초기 후보 지표:

- 매출액
- 영업이익
- 당기순이익
- 자산총계
- 부채총계
- 자본총계
- 영업활동현금흐름
- 투자활동현금흐름

파생 지표 후보:

- 매출성장률
- 영업이익성장률
- 영업이익률
- 순이익률
- ROE
- ROA
- 부채비율
- 유동비율
- 영업현금흐름/매출
- CAPEX 성향

### 5.3 분석 레이어

WICS 대분류별 기준선을 만들고 기업의 상대적 위치를 계산한다.

- 섹터별 평균
- 섹터별 중앙값
- 섹터별 분위수
- 기업별 percentile rank
- 기업별 상대점수

## 6. 비교 방식

섹터 비교는 평균만 사용하지 않는다. 이상치가 많을 수 있으므로 다음 기준을 우선 검토한다.

- 중앙값
- 사분위수
- IQR
- percentile rank
- z-score

1차 분석에서는 `중앙값 + 분위수 기반 상대위치`를 기본 비교 방식으로 사용한다.

## 7. 추진 단계

### Step 1. 데이터 점검

- WICS 기업 매핑 파일과 DART 회사 파일의 공통 키 확인
- 재무제표 CSV 컬럼 구조 확인
- 연도별 결측치와 중복 여부 점검

### Step 2. 분석용 마스터 테이블 설계

- 기업-연도 단위 테이블 정의
- WICS 대분류와 재무제표 연결 방식 확정
- 연결 실패 기업 처리 기준 정의

### Step 3. 핵심 재무지표 정의

- 필수 지표 선정
- 계산 공식 정리
- 연결/별도 재무 기준 통일 여부 결정

### Step 4. 섹터 기준선 산출

- WICS 대분류별 평균, 중앙값, 분위수 계산
- 섹터별 대표 지표 테이블 작성

### Step 5. 기업 상대평가 설계

- 각 기업의 섹터 내 위치 계산
- 성장성, 수익성, 안정성별 서브 스코어 설계
- 종합 점수 산정 방식 초안 작성

### Step 6. 텍스트 결합 계획

- 사업개요, 사업부문, 원재료, 리스크 관련 텍스트 수집 여부 검토
- 섹터별 핵심 키워드 추출 방안 설계
- 정량 지표와 서술 정보 연결 방식 정리

## 8. 초기 산출물

처음부터 아래 3개 결과물을 목표로 한다.

- WICS 대분류별 재무 특성표
- 기업별 섹터 상대평가 테이블
- 분석 해석 노트 또는 리포트 초안

## 9. 우선순위

단년도 분석이 정리되면 이후 아래 순서로 확장한다.

1. 2025년 WICS 대분류 기준선 구축
2. 2025년 기업 상대평가 구축
3. 2021~2025 시계열 확장
4. 사업보고서 텍스트 결합

## 10. 주의사항

- WICS 대분류만으로는 일부 업종 차이가 과도하게 뭉개질 수 있다.
- 금융, 지주, 리츠 성격 기업은 일반 제조업 지표와 동일 비교가 왜곡될 수 있다.
- 연결/별도 재무 혼용 여부를 초기에 정리해야 한다.
- 연도별 기준 변경이나 공시 누락 기업에 대한 예외처리가 필요하다.
