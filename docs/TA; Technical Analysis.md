# 📈 주식 자동매매 시스템 설계를 위한 5대 핵심 지표 가이드

본 문서는 자동매매 알고리즘의 **진입(Entry)**, **필터링(Filtering)**, **리스크 관리(Risk Management)** 로직을 설계할 때 필수적으로 고려해야 할 5가지 핵심 지표와 시스템 설계 원칙을 정리한 가이드입니다.

---

## 1. Bollinger Bands (볼린저 밴드) — 변동성 지표
- **원리:** 중심선(20일 이동평균) 위아래로 표준편차(σ) 2배씩 밴드를 형성. 가격의 통계적 범위를 측정합니다.
- **자동매매 활용:**
    - 가격이 하단 밴드 터치 → 매수 고려 (역추세/눌림목 전략)
    - 가격이 상단 밴드 터치 → 매도 고려
    - 밴드 폭이 급격히 좁아질 때(Squeeze) → 곧 큰 변동성이 발생할 것임을 예고
- **⚠️ 조심할 것:** 강한 추세장에서는 가격이 밴드를 타고 계속 이동합니다. 이때 하단 터치 시 매수하는 공식은 큰 손실로 이어질 수 있으므로, **ADX로 추세 강도를 확인**하는 것이 필수입니다.

## 2. RSI (상대강도지수) — 모멘텀 지표
- **원리:** 최근 N일 동안의 상승폭과 하락폭의 비율을 0~100으로 수치화하여 에너지의 강도를 측정합니다.
- **자동매매 활용:**
    - **30 이하:** 과매도 → 매수 고려
    - **70 이상:** 과매수 → 매도 고려
    - **Divergence:** 주가는 신고가인데 RSI는 낮아질 때 → 강력한 추세 반전 신호
- **⚠️ 조심할 것:** RSI가 30에 닿았다고 무조건 반등하지 않습니다. 하락 추세가 강하면 20대에서도 계속 빠질 수 있습니다. **횡보장에서 가장 잘 작동**하며 추세장에서는 신뢰도가 떨어집니다.

## 3. MACD — 추세 및 모멘텀 지표
- **원리:** 단기(12일)와 장기(26일) 지수이동평균(EMA)의 차이를 이용하여 추세의 전환점과 강도를 포착합니다.
- **자동매매 활용:**
    - **Golden Cross:** MACD선이 시그널선을 상향 돌파 → 매수 신호
    - **Dead Cross:** MACD선이 시그널선을 하향 돌파 → 매도 신호
    - **0선 돌파:** MACD선이 0선 위로 올라오면 상승 모멘텀 강화로 판단
- **⚠️ 조심할 것:** 대표적인 후행 지표입니다. 신호가 나올 때 이미 가격의 상당 부분이 진행된 경우가 많으며, **변동성이 심한 종목에서는 허위 신호(Whipsaw)**가 많이 발생합니다.

## 4. ADX (평균방향지수) — 추세 강도 지표
- **원리:** 현재 시장이 한 방향으로 얼마나 강하게 움직이는지 0~100으로 수치화합니다. (방향은 알려주지 않음)
- **자동매매 활용:**
    - **25 미만:** 추세 없음 → 볼린저 밴드, RSI 등 횡보장 전략 사용
    - **25 이상:** 추세 형성 → MACD 등 추세 추종 전략의 신뢰도 상승
    - **40 이상:** 강한 추세 → 추세 추종 전략 유효
- **⚠️ 조심할 것:** ADX는 방향을 알려주지 않습니다. 급락장에서도 ADX는 높게 나타날 수 있으므로 반드시 **+DI, -DI**와 함께 보거나 이동평균선으로 방향을 확인해야 합니다.

## 5. OBV (On-Balance Volume) — 거래량 지표
- **원리:** 가격 상승일의 거래량은 더하고, 하락일은 빼서 누적 계산합니다. 거래량이 가격에 선행한다는 원리를 이용합니다.
- **자동매매 활용:**
    - 가격 횡보 중 OBV 상승 → 세력의 매집 의심, 상승 가능성 높음
    - 가격 상승 중 OBV 하락 → 거래량 없는 상승, 하락 전환 예고
    - OBV 방향과 가격 방향 일치 → 현재 추세의 신뢰도 높음
- **⚠️ 조심할 것:** OBV는 절대 수치보다 **방향과 기울기**가 중요합니다. 주식 분할이나 블록딜 같은 이벤트로 인해 데이터가 왜곡될 수 있음을 주의해야 합니다.

## 🛠 자동매매 시스템 설계 시 3대 핵심 원칙

성공적인 알고리즘 구축을 위해 아래의 기술적 고려사항을 반드시 포함해야 합니다.

### 1. 지표의 조합과 필터링 (Multi-Indicator)
단일 지표는 속임수가 많습니다. 아래와 같은 단계적 필터링이 권장됩니다.
- **Step 1:** ADX로 추세 유무 판단 (Filter)
- **Step 2:** MACD로 진입 방향 결정 (Direction)
- **Step 3:** RSI 또는 OBV로 진입 확정 (Confirmation)

### 2. 타임프레임 및 데이터 확정 (Execution)
- **멀티 타임프레임:** 일봉으로 큰 추세를 확인하고, 15분봉이나 30분봉으로 세부 진입 시점을 잡는 것이 노이즈를 줄이는 방법입니다.
- **캔들 마감 기준:** 실시간(Tick) 데이터는 봉이 완성되기 전 신호가 나타났다 사라질 수 있습니다. 가급적 **봉 마감(Close)** 데이터를 기준으로 로직을 실행하여 허위 신호를 방지하세요.

### 3. 리스크 관리 로직 (Exit Strategy)
자동매매에서 가장 중요한 것은 '잘 사는 것'보다 '잘 파는 것'입니다.
- **Stop-loss:** 진입가 대비 일정 % 하락 시 기계적 손절.
- **Trailing Stop:** 수익이 발생함에 따라 익절 기준선을 높여가며 수익을 보존.
- **Position Sizing:** 전체 자산 대비 투입 비중을 조절하여 파산을 방지.

# 📈 주식 자동매매 시스템 설계를 위한 5대 핵심 지표 가이드 (실전 강화 버전)

본 문서는 자동매매 알고리즘의 **진입(Entry)**, **필터링(Filter)**, **리스크 관리(Risk Management)**를 실제로 구현 가능한 수준으로 설계하기 위한 가이드입니다.

---

# 📊 1. 핵심 지표 (Core Indicators)

## 1. Bollinger Bands — 변동성 지표

* 중심선: 20일 이동평균
* 밴드: ±2σ (표준편차)

### 활용

* 하단 터치 → 눌림목 매수
* 상단 터치 → 과열 매도
* 밴드 수축(Squeeze) → 변동성 폭발 전조

### 주의

* 추세장에서는 밴드를 따라 움직임 (밴드 워킹)
  → 반드시 ADX와 함께 사용

---

## 2. RSI — 모멘텀 지표

* 범위: 0 ~ 100

### 활용

* RSI < 30 → 과매도
* RSI > 70 → 과매수
* Divergence → 추세 반전 신호

### 주의

* 추세장에서는 신뢰도 낮음
* 횡보장에서 효과적

---

## 3. MACD — 추세 + 모멘텀

* 구성: EMA(12) - EMA(26)

### 활용

* Golden Cross → 매수
* Dead Cross → 매도
* 0선 돌파 → 추세 강화

### 주의

* 후행 지표
* 변동성 큰 시장에서 Whipsaw 발생

---

## 4. ADX — 추세 강도

* 범위: 0 ~ 100

### 활용

* ADX < 25 → 횡보
* ADX > 25 → 추세
* ADX > 40 → 강한 추세

### 주의

* 방향 없음 → +DI / -DI 또는 MA 필요

---

## 5. OBV — 거래량

* 누적 거래량 기반

### 활용

* 가격 횡보 + OBV 상승 → 매집
* 가격 상승 + OBV 하락 → 약한 상승
* 방향 일치 → 신뢰도 상승

---

# 🧠 2. 시장 상태 판단 (Regime Detection)

단순 ADX 하나로는 부족 → 반드시 결합 사용

## 권장 로직

```
if ADX > 25 and MA_slope > 0:
    상승 추세
elif ADX > 25 and MA_slope < 0:
    하락 추세
else:
    횡보
```

## 추가 옵션

* Bollinger Band Width
* ATR (변동성)

---

# 🎯 3. 진입 전략 (Entry Rules)

❌ 기존 문제: “고려” 수준
✅ 개선: 반드시 **수식 기반 조건**

## 예시 (추세 추종 전략)

```
매수 조건:
- ADX > 25
- MACD > Signal (최근 3봉 내 골든크로스)
- RSI 40 ~ 60
- 가격 > 20MA

→ 모든 조건 충족 시 진입
```

## 예시 (횡보 전략)

```
매수 조건:
- ADX < 25
- 가격이 Bollinger 하단 터치
- RSI < 30

→ 반등 기대 매수
```

---

# ⛔ 4. 청산 전략 (Exit Rules) — 핵심

자동매매에서 가장 중요한 요소

## 1. 손절 (Stop-loss)

```
손절 = entry_price - (ATR * 1.5)
```

## 2. 익절 (Take Profit)

```
익절 = entry_price + (ATR * 3)
```

## 3. 트레일링 스탑

* 수익 발생 시 손절선 상향

## 4. 신호 기반 청산

```
- MACD Dead Cross
- RSI > 70 후 하락
```

## 5. 시간 기반 청산

```
- n봉 경과 시 강제 청산
```

---

# 💰 5. 포지션 사이징 (Position Sizing)

❗ 계좌 생존에 가장 중요한 요소

## Fixed Fractional 방식

```
계좌 리스크 = 전체 자산의 1%

position_size = risk / (entry_price - stop_loss)
```

---

# 🕒 6. 멀티 타임프레임 전략

## 구조

* 상위 타임프레임 → 방향
* 하위 타임프레임 → 진입

## 예시

```
일봉:
- ADX > 25 → 추세 확인

15분봉:
- MACD 골든크로스 → 진입
```

---

# ⚙️ 7. 실행 로직 (Execution)

## 필수 원칙

* 캔들 마감 기준 사용
* 실시간 신호 금지 (노이즈 발생)

---

# 💸 8. 거래 비용 반영 (현실 반영)

## 반드시 포함

* 수수료
* 슬리피지
* 호가 단위

## 예시

```
실제 수익 = 전략 수익 - (수수료 + 슬리피지)
```

---

# 🧪 9. 백테스트 (Backtesting)

## 필수 검증 단계

### 1. 데이터 분리

* Train / Test

### 2. Walk-forward 테스트

* 구간별 반복 검증

### 3. 다양한 종목 적용

* 특정 종목 과최적화 방지

---

# 🧩 10. 최종 전략 구조 (추천 아키텍처)

```
1. Regime 판단 (ADX + MA slope)
2. 방향 결정 (MACD)
3. 진입 확인 (RSI / OBV)
4. 포지션 사이징
5. 진입
6. Exit (손절 / 익절 / 신호 / 시간)
```
