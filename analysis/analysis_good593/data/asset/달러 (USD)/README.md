## 달러 인덱스의 경제적 의미

달러 인덱스(DXY)는 **유로(57.6%), 엔(13.6%), 파운드(11.9%), 캐나다달러(9.1%),  
스웨덴크로나(4.2%), 스위스프랑(3.6%)** 6개 통화 대비 달러 가치를 가중 평균한 지수다.

원자재는 국제적으로 달러로 거래되므로 달러 강세 시 원자재 가격이 하락하고,  
달러 약세 시 원자재 가격이 상승하는 **역상관 관계**가 형성된다.  
한국처럼 수출 비중이 높은 국가의 기업 이익은 원/달러 환율을 통해 직접 영향을 받는다.

### 가격 구동 요인

| 요인 | DXY 방향 | 설명 |
|------|-----------|------|
| 미국 금리(TNX) 상승 | ↑ | 달러 자산 수익률 상승 → 달러 수요 증가 |
| 미국 경제 상대적 강세 | ↑ | 자본 유입 → 달러 강세 |
| 글로벌 위험회피 심리 | ↑ | 안전통화 달러로 자금 쏠림 |
| Fed 긴축(금리 인상) | ↑ | 달러 가치 상승 |
| Fed 완화(금리 인하) | ↓ | 달러 가치 하락 |
| 미국 무역적자 확대 | ↓ | 달러 공급 증가 → 가치 하락 |

## 해석 방법

### 가격 수준별 해석

| Close 범위 | 시장 해석 |
|------------|-----------|
| 110 이상 | 달러 초강세 — 신흥국 통화 압박, 원자재 급락 |
| 100 ~ 110 | 달러 강세 구간 — 원자재 약세 압력 |
| 90 ~ 100 | 달러 중립 구간 |
| 90 이하 | 달러 약세 — 원자재·신흥국 자산 강세 |

### 가격 방향성별 해석

| 신호 | 연쇄 영향 |
|------|-----------|
| DXY MA 상향 (강세) | 금·구리·WTI 하락 압력 / 수출 기업 환차익 |
| DXY MA 하향 (약세) | 금·구리·WTI 상승 / 신흥국 자본 유입 |
| DXY 급등 (단기 급등) | 글로벌 위험회피 극대화 → 전 자산 변동성 확대 |
| DXY 하락 + CPI 상승 | 달러 약세 + 인플레이션 동반 → 실물자산 강세 최적 환경 |
| DXY 상승 + TNX 상승 | 금리 인상 주도 달러 강세 → 성장주 동시 약세 |

### WICS 섹터별 영향

| 섹터 | 연관성 | 영향 |
|------|--------|------|
| 소재 | 높음 | 달러 강세 → 원자재 가격 하락 → 소재 기업 수익성 악화 |
| 산업재 | 중간 | 수출 기업 환율 영향 — 원화 약세(달러 강세) 시 수출 이익 |
| IT | 중간 | 해외 매출 비중 높을수록 달러 강세 시 환차익 |
| 금융 | 보통 | 통화정책 기대와 연동된 간접 영향 |

## 백테스팅 활용 (신호 전용)

DXY는 직접 매매 불가 — **다른 자산 매매 신호 생성에만 사용**

### 신호 생성 조합 (S-3 헷지 전략)

```
[달러 강세 → 원자재 비중 축소]
DXY Close > MA(20) → 금·구리 비중 축소, 현금·국채 확대

[달러 약세 → 실물자산 비중 확대]
DXY Close < MA(20) → 금·구리·WTI 비중 확대

[인플레이션 국면 판별]
DXY 약세 AND CPI_YoY >= 4% → 스태그플레이션 아닌 인플레이션 국면
                               → 실물자산 최우선 오버웨이트
```

### 주의사항
- `Volume`은 의미 없는 0 — 유동성 필터에 사용하지 말 것
- 이동평균은 Backtrader `bt.indicators.SMA`로 전략 내부에서 계산
