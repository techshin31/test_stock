## 금 가격의 경제적 의미

금은 **안전자산(Safe Haven)** 이자 **인플레이션 헷지 자산**의 대표 주자다.  
이자를 지급하지 않는 실물 자산이므로 기회비용(=금리)이 높을 때 불리하고,  
불확실성이 커지거나 달러 가치가 하락할 때 수요가 늘어난다.

### 가격 구동 요인

| 요인 | 가격 방향 | 설명 |
|------|-----------|------|
| 금리(TNX) 하락 | ↑ | 보유 기회비용 감소 → 금 매력도 상승 |
| 달러(DXY) 약세 | ↑ | 달러 표시 자산인 금의 가격 상승 |
| CPI 상승 (인플레이션) | ↑ | 실질 구매력 보존 수요 증가 |
| 지정학적 위험 확대 | ↑ | 위험회피 자금의 안전자산 유입 |
| 중앙은행 금 매입 증가 | ↑ | 달러 의존도 낮추기 위한 보유 확대 |
| 금리 급등 | ↓ | 기회비용 상승 → 금 매력도 하락 |

## 해석 방법

### 가격 수준별 해석

| Close 범위 | 시장 해석 |
|------------|-----------|
| $2,500/oz 이상 | 강한 위험회피 또는 달러 약세 국면 |
| $1,800 ~ $2,500/oz | 불확실성 고조, 인플레 헷지 수요 |
| $1,200 ~ $1,800/oz | 금리 안정, 중립 국면 |
| $1,200/oz 이하 | 강한 달러 또는 금리 인상기 |

### 가격 방향성별 해석

| 신호 | 해석 |
|------|------|
| 금 MA 상향 + TNX 하락 | 금리 완화 국면 → 금·채권 동반 강세 |
| 금 MA 상향 + DXY 하락 | 달러 약세 주도 금 상승 (전형적 패턴) |
| 금 MA 상향 + CPI_YoY ≥ 4% | 인플레이션 헷지 수요 → 실물자산 오버웨이트 |
| 금 하락 + TNX 상승 | 금리 인상 사이클 → 금 약세 정상 패턴 |
| 금 상승 + 주가 하락 | 위험회피 극대화 → 포트폴리오 방어 역할 확인 |

### WICS 섹터별 영향

| 섹터 | 연관성 | 영향 |
|------|--------|------|
| 금융 | 중간~높음 | 안전자산 선호 시 은행주 변동성 확대 |
| 소재 | 중간 | 금 채굴 기업(고려아연 등) 주가에 직접 영향 |
| 소비재 | 보통 | 금 가격 상승 = 위험회피 심리 → 소비 심리 위축 가능 |

## 백테스팅 활용

### 직접 투자 대상
`GC=F` 선물 데이터 — 금 ETF(예: GLD, IAU) 포지션의 대체 피드로 사용 가능

### 신호 생성 조합 (S-3 헷지 전략)

```
[달러 약세 → 금 비중 확대]
DXY Close < MA(20) → 금 비중 확대

[달러 강세 → 금 비중 축소]
DXY Close > MA(20) → 금 비중 축소, 현금 확대

[인플레이션 오버웨이트]
CPI_YoY >= 4% → 금·구리·WTI 실물자산 오버웨이트

[Risk-Off 안전자산]
BDI/구리 MA 하향 → 금 비중 확대
```

### 주의사항
- 이동평균(MA)·수익률은 Backtrader Indicator로 전략 내부에서 계산
- `MA20`, `MA60` 등 파생 컬럼 미포함 — 외부 사전 계산값은 룩어헤드 바이어스 위험
