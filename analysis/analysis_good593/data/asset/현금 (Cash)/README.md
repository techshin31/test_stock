## 1. 미국 10년 국채금리 (TNX)

### CSV 포맷

```
Date, Open, High, Low, Close, Volume
2024-01-02, 3.9690, 3.9730, 3.9180, 3.9460, 0
```

| 컬럼 | 설명 | 단위 |
|------|------|------|
| `Date` | 거래일 (YYYY-MM-DD, 오름차순) | — |
| `Open` | 당일 시가 금리 | % |
| `High` | 당일 최고 금리 | % |
| `Low` | 당일 최저 금리 | % |
| `Close` | 당일 종가 금리, **소수점 4자리** | % |
| `Volume` | **0 고정** (금리 지수 — 거래량 없음) | — |

### 경제적 의미

미국 10년 국채금리(TNX)는 **글로벌 장기 금리의 기준**이다.  
주식은 미래 이익을 현재 가치로 할인해 평가하는데, 이 **할인율**이 바로 국채금리다.  
금리가 오르면 미래 이익의 현재 가치가 낮아져 성장주 PER이 압박받고,  
금리가 내리면 반대로 성장주 밸류에이션이 확대된다.

연준(Fed)의 기준금리 결정에 가장 민감하게 반응하며,  
CPI(인플레이션)와 고용 지표에 의해 선행적으로 움직이는 경우가 많다.

### 가격 구동 요인

| 요인 | TNX 방향 | 설명 |
|------|-----------|------|
| CPI 예상 상회 | ↑ | 인플레이션 기대 상승 → 금리 인상 기대 |
| Fed 긴축 발언 | ↑ | 금리 인상 경로 확인 → 장기금리 상승 |
| 경기 침체 우려 | ↓ | 안전자산 국채 수요 증가 → 가격 상승, 금리 하락 |
| Fed 완화 전환 | ↓ | 금리 인하 기대 → 장기금리 하락 |
| 미국 국채 대규모 발행 | ↑ | 공급 증가 → 국채 가격 하락, 금리 상승 |

### 해석 방법

| TNX Close | 시장 국면 해석 |
|-----------|--------------|
| 5% 이상 | 고금리 긴축 극단 — 성장주 밸류에이션 극도로 압박 |
| 4 ~ 5% | 고금리 환경 — 배당주·가치주 선호, 성장주 약세 |
| 2 ~ 4% | 중립 금리 구간 — 밸류에이션 부담 적음 |
| 2% 이하 | 저금리 환경 — 성장주·IT 섹터 밸류에이션 극대화 |

| 신호 | 해석 |
|------|------|
| TNX 상향 (상승) | 성장주(IT) 밸류에이션 압박, 금 하락, 달러 강세 |
| TNX 하향 (하락) | 성장주 밸류에이션 확대, 금 상승, 채권 가격 상승 |
| TNX 4% 상향 돌파 | Risk-Off 전환 신호 — 현금·국채 비중 확대 검토 |
| TNX 급락 | 경기침체 우려 또는 Fed 긴급 완화 → 금 급등 동반 경우 많음 |
| 장단기 역전 (2Y > 10Y) | 경기침체 선행 지표 (단기금리 > 장기금리) |

### WICS 섹터별 영향

| 섹터 | 연관성 | 영향 |
|------|--------|------|
| IT | 높음 | 할인율 직접 영향 — 금리 상승 시 고PER 성장주 급락 |
| 금융 | 높음 | 순이자마진(NIM) 변화 → 은행 수익성 직결 |
| 소재·산업재 | 중간 | 설비 투자 비용 변화, 경기 기대와 연동 |
| 유틸리티·부동산 | 높음 | 고배당 섹터 — 금리 상승 시 채권 대비 매력도 감소 |

### 백테스팅 활용 (신호 전용)

```
[성장주 축소 신호]
TNX Close > 4.0% AND 상승 추세 → IT·성장주 비중 축소, 현금 확대

[성장주 확대 신호]
TNX Close < MA(20) AND 하락 추세 → SOX·IT 섹터 비중 확대

[금과의 역상관 조합]
TNX 상승 AND 금 하락 → 정상 패턴 확인 (헷지 전략 점검)
TNX 하락 AND 금 상승 → Risk-Off 국면 심화 (비정상 패턴)
```

## 2. 미국 소비자물가지수 (CPI)

### CSV 포맷

```
Date, Close, CPI_YoY
2024-01-02, 308.148, 3.13
```

| 컬럼 | 설명 | 단위 |
|------|------|------|
| `Date` | 영업일 기준 (YYYY-MM-DD, 오름차순) | — |
| `Close` | CPI 지수값, **소수점 3자리**, forward-fill 적용 | 지수 |
| `CPI_YoY` | 전년 동기 대비 변화율, 소수점 2자리 | % |

### 데이터 처리 특이사항

| 처리 | 내용 |
|------|------|
| 월봉 → 일봉 확장 | 매월 발표값을 다음 발표일 전날까지 forward-fill |
| **발표 지연 15일 적용** | `observation_date + 15일`부터 해당 월 CPI 적용 |
| 초기 NaN → 0.0 | 앞 12개월 YoY 계산 불가 구간을 0으로 채움 |

> **룩어헤드 바이어스 방지:** 미국 CPI는 해당 월 기준 약 2주 후 발표  
> (예: 2024년 3월 CPI → 2024년 4월 16일부터 사용 가능)

### 경제적 의미

CPI(Consumer Price Index)는 도시 소비자가 구매하는 재화·서비스 가격 변동을 측정하는  
**인플레이션 지표**다. 연준(Fed)의 통화정책 목표는 **CPI_YoY 2%** 이며,  
이를 기준으로 금리 인상/인하 결정이 내려진다.

CPI 발표는 주식·채권·외환 시장 전반에 즉각적 영향을 미치며,  
특히 예상치를 상회하는 CPI는 금리 인상 기대를 높여 성장주에 부정적으로 작용한다.

### CPI_YoY 해석 방법

| CPI_YoY | 국면 | 통화정책 기대 | 전략 방향 |
|---------|------|--------------|-----------|
| 6% 이상 | 과열 인플레이션 | 연준 급격한 금리 인상 | 실물자산 최우선, 성장주 회피 |
| 4 ~ 6% | 높은 인플레이션 | 지속적 금리 인상 | 금·구리·WTI 오버웨이트 |
| 2 ~ 4% | 골디락스 (적정) | 금리 안정 또는 소폭 조정 | 균형 포트폴리오 유지 |
| 1 ~ 2% | 적정 인플레이션 | 금리 동결 또는 인하 준비 | 성장주·IT 섹터 선호 |
| 1% 이하 | 디플레이션 우려 | 금리 인하, 양적완화 | 국채·현금 비중 확대 |

| 신호 조합 | 해석 |
|-----------|------|
| CPI_YoY ≥ 4% + DXY 약세 | 인플레이션 국면 → 금·구리·WTI 실물자산 오버웨이트 |
| CPI_YoY ≥ 4% + TNX 상승 | 연준 긴축 사이클 → 성장주 축소, 현금 확대 |
| CPI_YoY < 2% + TNX 하락 | 경기침체 우려 → 국채·금 선호 |
| CPI_YoY 2~3% + 구리·BDI 상승 | 골디락스 + 경기 확장 → Risk-On 최적 환경 |

### WICS 섹터별 영향

| 섹터 | 연관성 | 영향 |
|------|--------|------|
| IT | 높음 | 금리 상승 기대 → 고PER 성장주 밸류에이션 압박 |
| 소재·에너지 | 높음 | 원자재 가격 상승이 CPI를 견인 — 동반 강세 |
| 필수소비재 | 중간 | 가격 전가 능력 있는 기업 방어적 역할 |
| 금융 | 중간~높음 | 금리 인상 수혜 (은행 NIM 확대) |

### 백테스팅 활용 (신호 전용)

```
[인플레이션 국면 — 실물자산 오버웨이트]
CPI_YoY >= 4.0 → 금·구리·WTI 비중 확대

[골디락스 국면 — Risk-On 유지]
2.0 <= CPI_YoY < 4.0 AND TNX 안정 → 균형 포트폴리오, IT 비중 유지

[디플레이션 우려 — 방어 자산 전환]
CPI_YoY < 2.0 → 국채·현금 비중 확대
```
