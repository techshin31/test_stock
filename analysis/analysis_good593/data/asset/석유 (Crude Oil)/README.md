## WTI 원유 가격의 경제적 의미

WTI(West Texas Intermediate)는 미국산 경질 저유황 원유로 **글로벌 원유 가격의 기준**이다.  
산업 전반의 에너지 비용을 결정하며 경기 싸이클과 밀접하게 연동된다.  
적당한 유가 상승은 경기 회복 수요를 반영하지만, 급등은 인플레이션을 자극해  
소비·기업 이익 양쪽을 동시에 압박하는 **양날의 검** 역할을 한다.

### 가격 구동 요인

| 요인 | 가격 방향 | 설명 |
|------|-----------|------|
| OPEC+ 감산 결정 | ↑ | 공급 축소 → 가격 지지 |
| 글로벌 경기 회복 | ↑ | 수요 증가 |
| 달러(DXY) 약세 | ↑ | 달러 표시 원자재 상승 |
| 미국 셰일 증산 | ↓ | 공급 증가 → 가격 하락 압력 |
| 경기 침체 우려 | ↓ | 수요 감소 전망 |
| 전략비축유(SPR) 방출 | ↓ | 단기 공급 확대 |

## 해석 방법

### 가격 수준별 해석

| Close 범위 | 시장 해석 |
|------------|-----------|
| $100/배럴 이상 | 공급 충격 또는 지정학적 위기 — 인플레이션 압박 극대화 |
| $70 ~ $100/배럴 | 경기 확장 국면, 수요 양호 |
| $40 ~ $70/배럴 | 경기 중립 또는 수요 둔화 |
| $40/배럴 이하 | 경기 침체 또는 공급 과잉 (2020년 코로나 수준) |

### 가격 방향성별 해석

| 신호 | 해석 |
|------|------|
| WTI 완만한 상승 | 경기 회복 수요 증가 → 에너지·산업재 섹터 긍정 |
| WTI 급등 (단기 급등) | 공급 충격 → 인플레이션 압박, 소비·기업 이익 동시 압박 |
| WTI 하락 | 수요 감소(경기 둔화) 또는 공급 과잉 |
| WTI + 구리 동반 상승 | 글로벌 수요 강도 확인 → 강한 Risk-On 신호 |
| WTI + 구리 동반 하락 | 글로벌 수요 위축 → 경기 침체 선행 신호 |
| DXY 강세 + WTI 하락 | 달러-원유 역상관 패턴 (정상 반응) |

### WICS 섹터별 영향

| 섹터 | 연관성 | 영향 |
|------|--------|------|
| 에너지 | 매우 높음 | 유가 상승 = 직접 수익성 개선, 탐사·채굴 투자 증가 |
| 소재 | 높음 | 에너지 비용 상승 → 화학·철강 원가 부담 |
| 산업재 | 높음 | 물류·운송 비용 증가 → 수익성 압박 |
| 소비재 | 낮음~중간 | 휘발유 가격 → 가처분 소득 감소 → 소비 위축 |
| IT·금융 | 낮음 | 간접 영향 위주 |

## 백테스팅 활용

### 직접 투자 대상
`CL=F` 선물 데이터 — 원유 ETF(예: USO, BNO) 포지션의 대체 피드로 사용 가능

### 신호 생성 조합

```
[인플레이션 헷지]
CPI_YoY >= 4% AND WTI MA 상향 → WTI(원유) 비중 확대

[경기 강도 확인]
WTI MA 상향 AND 구리 MA 상향 → 강한 경기 확장 → 산업재·소재 편입

[에너지 섹터 진입]
WTI Close > MA(60) → 에너지 섹터 비중 확대

[Risk-Off]
WTI MA 하향 AND 구리 MA 하향 → 경기 둔화 → 방어 자산 전환
```

### 주의사항
- `Volume`은 CL=F 실거래량 — 유동성 필터에 활용 가능
- 이동평균·수익률 등 파생 컬럼 미포함 — 전략 내부 계산 필수
- WTI 급락 시 음수 Close 가능성 주의 (2020년 4월 역대 -37달러 사례)
