## 공통 CSV 포맷

```
Date, Open, High, Low, Close, Volume
2024-01-02, ..., ..., ..., ..., ...
```

- `Date`: `YYYY-MM-DD` 형식, **오름차순** 정렬
- `Volume`: 거래량이 없는 지수·금리 자산은 `0` 고정
- 파생 컬럼(이동평균, 일일수익률 등) **미포함** — Backtrader Indicator로 전략 내부 계산

## 1. 구리 (Copper)

**파일:** `copper_{year}.csv` | **소스:** Yahoo Finance `HG=F` (구리 선물) | **단위:** USD/lb, 소수점 4자리

### 경제적 의미

구리는 전선·모터·건설·전자 등 산업 전반에 쓰이는 핵심 원자재다.  
글로벌 경제 활동이 활발할수록 수요가 늘어 가격이 오르기 때문에  
**"닥터 쿠퍼(Dr. Copper)"** 라는 별명처럼 경기의 선행 지표로 널리 활용된다.

### 해석 방법

| 신호 | 해석 |
|------|------|
| 구리 가격 **상승** | 글로벌 제조업 수요 증가, 경기 확장 국면 진입 가능 |
| 구리 가격 **하락** | 수요 위축, 경기 둔화 또는 침체 우려 |
| 구리 + BDI **동반 상승** | Risk-On 신호 → 산업재·소재 섹터 비중 확대 |
| 구리 + BDI **동반 하락** | Risk-Off 신호 → 금·국채 비중 확대 |
| 달러 강세 + 구리 하락 | 달러-원자재 역상관 관계 확인 |

### WICS 섹터 연관성

- **소재 (높음):** 구리 원가 직접 영향 — 가격 상승 시 원가 부담 증가
- **산업재 (높음):** 설비 투자·건설 경기와 연동
- **IT (보통):** 반도체·전자부품 제조 원료로 간접 영향

### 백테스팅 활용

- **직접 투자:** HG=F 선물 추종 ETF(예: CPER) 대용으로 사용
- **신호 생성:** BDI와 조합해 Risk-On/Off 판별

## 2. 금 (Gold)

**파일:** `gold_{year}.csv` | **소스:** Yahoo Finance `GC=F` (금 선물) | **단위:** USD/oz, 소수점 2자리

### 경제적 의미

금은 대표적 **안전자산(Safe Haven)** 이자 **인플레이션 헷지 자산**이다.  
달러 인덱스(DXY)와 역상관 경향이 강하고, 금리가 오르면 기회비용 상승으로 금 수요가 줄어드는 특성이 있다.

### 해석 방법

| 신호 | 해석 |
|------|------|
| 금 가격 **상승** | 위험회피 심리 강화, 인플레 기대 상승, 달러 약세 |
| 금 가격 **하락** | 위험선호 심리 회복, 금리 상승, 달러 강세 |
| CPI_YoY ≥ 4% + 금 상승 | 인플레이션 국면 — 실물자산 오버웨이트 신호 |
| TNX 하락 + 금 상승 | 금리 하락기 안전자산 선호 강화 |
| DXY 하락 + 금 상승 | 달러 약세 → 금 강세 역상관 패턴 확인 |

### WICS 섹터 연관성

- **금융 (중간~높음):** 안전자산 선호 심리 변화로 금융주 변동성에 영향
- **소재 (중간):** 금 채굴 기업 주가에 직접 영향

### 백테스팅 활용

- **직접 투자:** GC=F 추종 ETF(예: GLD) 대용
- **신호 생성:** S-3 헷지 전략의 핵심 자산  
  - 달러 약세(DXY MA 하향) → 금 비중 확대  
  - CPI_YoY ≥ 4% → 금·구리·원유 실물자산 오버웨이트

## 3. 달러 인덱스 (DXY)

**파일:** `dxy_{year}.csv` | **소스:** Yahoo Finance `DX-Y.NYB` (ICE Dollar Index) | **단위:** 지수 포인트, 소수점 4자리  
**Volume:** 0 고정 (현물 지수)

### 경제적 의미

달러 인덱스는 **유로·엔·파운드·캐나다달러·스웨덴크로나·스위스프랑** 6개 통화 대비 달러 가치를 나타낸다.  
원자재는 달러로 거래되므로 달러 강세 시 원자재 가격이 하락하는 역상관 관계가 형성된다.  
한국처럼 수출 비중이 높은 국가의 기업 이익에도 환율을 통해 직접 영향을 미친다.

### 해석 방법

| DXY 방향 | 연쇄 영향 |
|-----------|-----------|
| **강세** (MA 상향) | 금·구리·원유 가격 하락 압력 → 소재·에너지 섹터 부담 |
| **약세** (MA 하향) | 원자재 가격 상승 → 실물자산 수혜, 신흥국 통화 강세 |
| **급등** | 신흥국 달러 부채 상환 부담 증가 → 글로벌 긴축 우려 |

### WICS 섹터 연관성

- **소재 (높음):** 달러 강세 → 원자재 가격 하락 → 수익성 악화
- **산업재·IT (중간):** 해외 매출 비중에 따라 환율 영향
- **금융 (보통):** 금리·통화정책 변화와 결합된 간접 영향

### 백테스팅 활용 (신호 전용)

- 달러 강세(DXY 20MA 상향) → 금·구리 비중 **축소**, 현금·국채 비중 확대
- 달러 약세(DXY 20MA 하향) → 금·구리·원유 비중 **확대**
- CPI와 조합: DXY 약세 + CPI_YoY ≥ 4% → 인플레이션 국면 판별

## 4. WTI 원유

**파일:** `wti_{year}.csv` | **소스:** Yahoo Finance `CL=F` (WTI 원유 선물) | **단위:** USD/barrel, 소수점 2자리

### 경제적 의미

WTI(West Texas Intermediate)는 글로벌 원유 가격의 기준이 되는 지표다.  
산업 전반의 에너지 비용을 결정하며, 적당한 상승은 경기 개선 신호지만  
급등은 인플레이션 압력과 기업 원가 부담으로 이어져 주식시장에 부정적 영향을 준다.

### 해석 방법

| 신호 | 해석 |
|------|------|
| WTI **완만한 상승** | 경기 회복 수요 증가 → 에너지·산업재 섹터 긍정 |
| WTI **급등** | 인플레이션 압박, 소비 위축 우려 → 광범위한 섹터 부담 |
| WTI **하락** | 수요 감소(경기 둔화) 또는 공급 과잉 |
| WTI + 구리 **동반 상승** | 글로벌 수요 강도 확인 → 강한 Risk-On 신호 |
| DXY 강세 + WTI 하락 | 달러-원유 역상관 패턴 |

### WICS 섹터 연관성

- **에너지 (매우 높음):** 유가 상승 시 직접 수익성 개선
- **소재·산업재 (높음):** 에너지 원가 부담 변수
- **소비재·IT (낮음):** 간접 비용 영향

### 백테스팅 활용

- **직접 투자:** CL=F 추종 ETF(예: USO) 대용
- CPI_YoY ≥ 4% 국면에서 인플레 헷지 자산으로 비중 확대
- BDI·구리와 함께 글로벌 수요 강도 확인 지표

## 5. 필라델피아 반도체 지수 (SOX)

**파일:** `sox_{year}.csv` | **소스:** Yahoo Finance `^SOX` | **단위:** 지수 포인트, 소수점 2자리  
**Volume:** 0 고정 (지수 — 직접 거래 불가)

### 경제적 의미

필라델피아 반도체 지수(PHLX Semiconductor Index)는 엔비디아·TSMC·인텔·퀄컴 등  
주요 반도체 기업 30개로 구성된 섹터 지수다.  
메모리(DRAM·NAND) 가격과 함께 IT 섹터 경기의 선행 지표로 활용된다.  
AI·데이터센터 수요 확대로 최근 글로벌 증시에서 핵심 지수로 부상했다.

### 해석 방법

| 신호 | 해석 |
|------|------|
| SOX **상승** | 반도체 수요 확대, IT 섹터 강세, Risk-On 분위기 |
| SOX **하락** | 반도체 재고 조정, 수요 둔화, 위험회피 심리 |
| SOX + 구리 **동반 상승** | IT·소재 섹터 동반 강세 → 강한 Risk-On 신호 |
| TNX 상승 + SOX 하락 | 금리 인상기 성장주 밸류에이션 압박 |
| TNX 하락 + SOX 상승 | 금리 완화 → IT·성장주 밸류에이션 확대 |

### WICS 섹터 연관성

- **IT (매우 높음):** 반도체는 IT 섹터 핵심 — 삼성전자·SK하이닉스 주가와 직결
- **소재 (보통):** 반도체 제조용 화학물질·특수가스 수요 반영
- **산업재 (낮음):** 반도체 장비 기업 간접 영향

### 백테스팅 활용 (신호 전용)

- SOX MA 상향 + 구리 MA 상향 → IT·소재 섹터 비중 확대
- SOX MA 하향 + TNX 상승 → 성장주 비중 축소, 가치주·배당주 전환

## 6. BDRY ETF / BDI 해운 운임

**파일:** `bdry_{year}.csv` | **소스:** Yahoo Finance `BDRY` (Breakwave Dry Bulk Shipping ETF) | **단위:** USD/주, 소수점 2자리

### 경제적 의미

**BDI(Baltic Dry Index)** 는 철광석·석탄·곡물 등 건화물 운임 비용을 반영하는 지수로,  
원자재 물동량과 글로벌 교역 수준의 선행 지표다.  
**BDRY ETF** 는 BDI를 추종하는 상장 ETF로, BDI 원지수와 높은 상관관계를 가지면서도 yfinance로 자동 수집·직접 투자가 가능하다.

> BDI 원지수가 필요한 경우: `collect_shipping.py --mode stooq --apikey YOUR_KEY`

### 해석 방법

| 신호 | 해석 |
|------|------|
| BDRY(BDI) **상승** | 원자재 물동량 증가, 글로벌 교역 활성화 → 경기 확장 선행 |
| BDRY(BDI) **하락** | 물동량 감소, 수요 위축 → 경기 둔화 선행 |
| BDI + 구리 **동반 상승** | 강한 Risk-On → 산업재·소재 편입 신호 |
| BDI + 구리 **동반 하락** | 강한 Risk-Off → 금·국채 편입 신호 |
| SCFI 상승 | 완제품(컨테이너) 물동량 증가 → 소비재·IT 섹터 수요 강도 보조 신호 |

### WICS 섹터 연관성

- **산업재 (매우 높음):** 해운·물류 기업 수익성과 직결
- **소재 (높음):** 철강·화학 원자재 수출입 물동량 반영
- **IT·소비재 (낮음):** 컨테이너(SCFI)가 더 관련성 높음

### 백테스팅 활용

- **직접 투자(BDRY):** 해운 업황 상승기 포지션 진입 가능
- **신호 생성:** BDI/BDRY + 구리 조합으로 S-2 매크로 국면 전략의 핵심 신호

## 7. 미국 10년 국채금리 (TNX)

**파일:** `tnx_{year}.csv` | **소스:** Yahoo Finance `^TNX` | **단위:** % (금리), 소수점 4자리  
**Volume:** 0 고정 (금리 지수 — 직접 거래 불가)

### 경제적 의미

미국 10년 국채금리는 **글로벌 장기 금리의 기준**이자 **할인율**이다.  
주식 밸류에이션(특히 성장주)은 미래 현금흐름을 현재 가치로 할인하는 방식으로 산출되기 때문에  
금리가 오르면 성장주 밸류에이션이 낮아지고, 반대로 금리가 내리면 높아진다.  
또한 국채금리는 경기 기대와 인플레이션 전망을 반영하는 신호등 역할을 한다.

### 해석 방법

| 금리 방향 | 연쇄 영향 |
|-----------|-----------|
| **상승** (hawkish) | 성장주(IT) 밸류에이션 압박, 금 가격 하락, 달러 강세 |
| **하락** (dovish) | 성장주 밸류에이션 확대, 금 가격 상승, 채권 가격 상승 |
| 2% 이하 | 저금리 환경 → 성장주·IT 섹터 강세 구간 |
| 4% 이상 | 고금리 부담 → 배당주·가치주 선호, 성장주 약세 |
| 장단기 역전 (2Y > 10Y) | 경기침체 선행 신호 (본 데이터에서 직접 계산 가능) |

### WICS 섹터 연관성

- **IT (높음):** 금리 민감도 가장 큼 — 할인율 직접 영향
- **금융 (높음):** 순이자마진(NIM) 변화로 수익성에 직접 영향
- **유틸리티·부동산 (높음):** 고배당 섹터 — 금리 상승 시 매력도 감소

### 백테스팅 활용 (신호 전용)

- TNX 상향 돌파(예: 4%) → Risk-Off, 성장주 축소, 현금 비중 확대
- TNX 하향 → SOX·IT 섹터 비중 확대 신호
- 금(Gold) 가격과 역상관 조합으로 포지션 조정

## 8. 미국 소비자물가지수 (CPI)

**파일:** `cpi_{year}.csv` | **소스:** FRED `CPIAUCSL` | **컬럼:** `Date, Close, CPI_YoY`

```
Date,Close,CPI_YoY
2024-01-02,308.148,3.13
```

- `Close`: CPI 지수값 (소수점 3자리)
- `CPI_YoY`: 전년 동기 대비 변화율 (%)

### 경제적 의미

CPI(Consumer Price Index)는 가계가 구매하는 재화·서비스 가격 변동을 측정하는 **인플레이션 지표**다.  
연준(Fed)의 물가 목표는 **2%** 이며, 이 수치를 기준으로 통화정책 방향이 결정된다.  
주식시장은 CPI 발표에 민감하게 반응하며, 특히 예상치를 상회하는 CPI는 금리 인상 기대를 높여 주가에 부정적으로 작용한다.

### 데이터 처리 특이사항

| 처리 | 내용 | 이유 |
|------|------|------|
| 월봉 → 일봉 forward-fill | 발표월 값을 다음 발표일까지 유지 | 일봉 전략 루프에서 CPI 값 단절 방지 |
| **발표 지연 15일 적용** | `observation_date + 15일`부터 적용 | 룩어헤드 바이어스 방지 — CPI는 익월 중순 발표 |
| 초기 NaN → 0.0 | 앞 12개월 YoY 계산 불가 구간 | Indicator 계산 오류 방지 |

### 해석 방법

| CPI_YoY | 국면 | 전략 방향 |
|---------|------|-----------|
| **< 2%** | 디플레이션 우려 | 국채·현금 비중 확대, 소비재 관망 |
| **2~4%** | 골디락스(적정 인플레) | Risk-On 유지, 균형 포트폴리오 |
| **≥ 4%** | 인플레이션 국면 | 금·구리·원유 실물자산 오버웨이트 |
| **≥ 6%** | 과열 인플레 | 연준 급격한 금리 인상 → 전 자산 변동성 확대 |

### 조합 신호

- `CPI_YoY ≥ 4%` + `DXY 약세` → 스태그플레이션 위험 → 금·원유 최우선
- `CPI_YoY ≥ 4%` + `TNX 상승` → 연준 긴축 국면 → 성장주 축소, 현금 확대
- `CPI_YoY < 2%` + `TNX 하락` → 경기침체 우려 → 국채·금 선호

## 자산 간 상관관계 요약

```
구리  ↑  →  산업재·소재 ↑  (경기 확장)
금    ↑  →  위험회피 심리 ↑  / DXY ↓, TNX ↓ 와 함께 움직임
DXY   ↑  →  금·구리·WTI ↓  (달러-원자재 역상관)
TNX   ↑  →  성장주(IT) ↓  / 금 ↓  (금리-성장주 역상관)
BDI   ↑  →  구리 수요 확인 → Risk-On
CPI   ↑  →  TNX ↑ 기대 → 연준 긴축 → 성장주 ↓, 실물자산 ↑
```

---

## 매크로 국면 판별 조합

| 국면 | 판별 조건 | 전략 |
|------|-----------|------|
| **Risk-On** | BDI·구리 MA 상향 + CPI 2~4% | 산업재·소재·IT 비중 확대 |
| **Risk-Off** | BDI·구리 MA 하향 + TNX 급등 | 금·국채 비중 확대, 현금 보유 |
| **인플레이션** | CPI_YoY ≥ 4% + DXY 약세 | 금·구리·WTI 오버웨이트 |
| **골디락스** | CPI 2~3% + TNX 안정 + SOX 상승 | 성장주(IT) 비중 확대 |
| **스태그플레이션** | CPI ↑ + 구리·BDI ↓ 동반 | 전 자산 변동성 확대, 방어적 배분 |
