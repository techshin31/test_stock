## 구리 가격의 경제적 의미

구리는 전선·모터·건설·전자제품 등 산업 전반에 쓰이는 핵심 원자재다.  
수요가 글로벌 경제 활동과 직결되기 때문에 **"닥터 쿠퍼(Dr. Copper)"** 라는 별명처럼  
경기 선행 지표로 널리 활용된다.

### 가격 구동 요인

| 요인 | 가격 방향 | 설명 |
|------|-----------|------|
| 글로벌 제조업 PMI 상승 | ↑ | 공장 가동률 증가 → 구리 수요 확대 |
| 중국 경기 부양 | ↑ | 세계 최대 소비국(전체 수요의 약 55%) |
| 달러 강세 | ↓ | 달러 표시 원자재 가격 하락 압력 |
| 생산국(칠레·페루) 파업·재해 | ↑ | 공급 충격 → 가격 급등 |
| 에너지 전환 수요 | ↑ | 전기차·태양광·풍력 인프라 확대 |

## 해석 방법

### 가격 수준별 해석

| Close 범위 | 시장 해석 |
|------------|-----------|
| $4.5/lb 이상 | 경기 과열 또는 공급 충격 국면 |
| $3.5 ~ $4.5/lb | 경기 확장 국면, 수요 양호 |
| $2.5 ~ $3.5/lb | 경기 중립 또는 완만한 둔화 |
| $2.5/lb 이하 | 경기 침체 우려, 수요 급감 |

### 가격 방향성별 해석

| 신호 | 해석 |
|------|------|
| 구리 MA 상향 돌파 | 글로벌 경기 회복 선행 신호 → Risk-On |
| 구리 MA 하향 이탈 | 경기 둔화 선행 신호 → Risk-Off |
| 구리 + BDI 동반 상승 | 수요·물동량 동시 증가 → 강한 경기 확장 신호 |
| 구리 + BDI 동반 하락 | 글로벌 교역 수요 위축 → 방어적 포트폴리오 전환 |
| 구리 상승 + 달러(DXY) 하락 | 달러 약세 → 원자재 가격 상승 정상 패턴 |
| 구리 상승 + 달러(DXY) 상승 | 수요 주도 강세 (더 강한 신호) |

### WICS 섹터별 영향

| 섹터 | 연관성 | 영향 |
|------|--------|------|
| 소재 | 매우 높음 | 구리 원가 직접 반영 — 가격 상승 시 원가 부담 |
| 산업재 | 높음 | 설비·건설 투자와 연동, 구리 수요 증가 = 수주 증가 |
| IT | 보통 | 반도체·전자부품 제조 원료, 직접 영향은 제한적 |
| 에너지 | 낮음 | 에너지 전환(전기차·태양광) 구조적 수요 간접 연결 |

## 백테스팅 활용

### 직접 투자 대상
`HG=F` 선물 데이터 — 구리 선물 추종 ETF(예: CPER, JJC) 포지션의 대체 피드로 사용 가능

### 신호 생성 조합

```
[Risk-On 신호]
구리 Close > MA(20) AND BDI/BDRY 상승 → 산업재·소재 섹터 편입

[Risk-Off 신호]
구리 Close < MA(20) AND BDI/BDRY 하락 → 금·국채 비중 확대

[인플레이션 헷지]
CPI_YoY >= 4% AND 구리 MA 상향 → 구리 비중 오버웨이트
```

### 주의사항
- `Volume`은 HG=F 선물 계약 수 — 유동성 필터링에 활용 가능
- 이동평균(MA)은 Backtrader `bt.indicators.SMA`로 전략 내부에서 계산
- 파생 컬럼(일일수익률 등)은 미포함 — 전략 내부에서 계산해야 룩어헤드 바이어스 방지
