## 필라델피아 반도체 지수의 경제적 의미

SOX는 **엔비디아·TSMC·퀄컴·인텔·브로드컴** 등 글로벌 주요 반도체 기업 30개로 구성된 섹터 지수다.  
반도체는 스마트폰·PC·서버·자동차·가전 등 모든 전자 제품의 핵심 부품으로,  
**IT 섹터 경기의 선행 지표**이자 글로벌 기술 수요의 온도계 역할을 한다.

AI·데이터센터 수요 폭증으로 2023년 이후 SOX의 글로벌 증시 내 영향력이 크게 확대됐으며,  
삼성전자·SK하이닉스의 DRAM·NAND 가격과도 높은 연동성을 보인다.

### 가격 구동 요인

| 요인 | SOX 방향 | 설명 |
|------|-----------|------|
| AI·서버 수요 증가 | ↑ | 엔비디아 GPU, HBM 수요 폭증 |
| PC·스마트폰 수요 회복 | ↑ | 메모리(DRAM·NAND) 가격 상승 연동 |
| 금리(TNX) 하락 | ↑ | 성장주 밸류에이션 확대 |
| 반도체 재고 조정 | ↓ | 과잉 공급 → 가격 하락 → 기업 이익 감소 |
| 금리 급등 | ↓ | 할인율 상승 → 성장주 PER 압박 |
| 미·중 반도체 규제 | ↓ | 수출 제한 → 매출 감소 우려 |

## 해석 방법

### 가격 수준별 해석

수준 자체보다 **방향성과 이동평균 대비 위치**가 중요하다.

| 신호 | 해석 |
|------|------|
| SOX > MA(20) > MA(60) | 반도체 섹터 강세 추세 확인 |
| SOX < MA(20) < MA(60) | 반도체 섹터 약세 추세 — IT 비중 축소 검토 |
| SOX 신고가 경신 | AI·데이터센터 사이클 확장 국면 |
| SOX 급락 후 반등 | 반도체 사이클 저점 탐색 → 중장기 매수 기회 |

### 가격 방향성별 해석

| 신호 | 해석 |
|------|------|
| SOX + 구리 동반 상승 | IT + 산업재 동반 강세 → 강한 경기 확장 + Risk-On |
| SOX 상승 + TNX 하락 | 금리 완화 → 성장주 밸류에이션 확대 (골디락스 국면) |
| SOX 하락 + TNX 상승 | 금리 인상기 성장주 압박 — IT 섹터 언더웨이트 |
| SOX 하락 + 구리 하락 | 광범위한 Risk-Off — 방어 자산 전환 시그널 |
| SOX 상승 + 달러 강세 | 미국 IT 기업의 달러 강세 수혜 (해외 매출 달러 환산 증가) |

### WICS 섹터별 영향

| 섹터 | 연관성 | 영향 |
|------|--------|------|
| IT | 매우 높음 | 삼성전자·SK하이닉스·삼성SDI 등 주가와 직결 |
| 소재 | 보통 | 반도체 제조용 특수가스·화학물질 수요 반영 |
| 산업재 | 낮음 | 반도체 장비 기업(ASML류) 간접 영향 |
| 소비재 | 낮음 | 스마트폰·PC 수요 변화를 통한 간접 영향 |

## 백테스팅 활용 (신호 전용)

SOX 지수는 직접 매매 불가 — **IT 섹터 편입 판단 신호로만 사용**  
(직접 투자 시 SOXX·SMH 등 반도체 ETF 활용)

### 신호 생성 조합

```
[IT 섹터 편입 신호]
SOX Close > MA(20) AND 구리 MA 상향 → IT·소재 섹터 비중 확대

[IT 섹터 축소 신호]
SOX Close < MA(20) AND TNX 상승 → 성장주 비중 축소, 가치주·배당주 전환

[골디락스 국면]
SOX 상승 AND TNX 안정(2~3%) AND CPI_YoY 2~4% → IT 섹터 오버웨이트
```

### 주의사항
- `Volume` 0 고정 — 유동성 필터 사용 불가, 전략 로직에서 Volume 조건 제외
- 이동평균·수익률 파생 컬럼 미포함 — 전략 내부 `bt.indicators.SMA`로 계산
