## 전략 지표 개요

### 방향형 전략 (언제 사고 팔지)

| 전략 | 유형 | 핵심 원리 | 적합 시장 |
|------|------|----------|----------|
| **골든크로스** | 추세 추종 | 단기 MA가 장기 MA를 상향 돌파 → 매수 | 강한 추세장 |
| **볼린저 밴드** | 평균 회귀 | 가격이 하단 밴드 돌파 후 회복 → 매수 | 횡보·박스권 |
| **RSI** | 역추세 | RSI < 30 과매도 해소 시 → 매수 | 횡보장 |
| **MACD** | 추세+모멘텀 | MACD선이 Signal선 상향 돌파 → 매수 | 추세 전환점 |

### 보조 지표 (신호 품질 향상용 필터)

#### ADX — 평균방향지수 (Average Directional Index)

**원리**: 현재 시장이 한 방향으로 얼마나 강하게 움직이는지를 0~100으로 수치화합니다.  
방향(상승/하락)은 알려주지 않고 **추세 강도**만 측정합니다.

```
ADX 계산 흐름:
  +DM (상승 방향 움직임) ─┐
                          ├─ TR(진폭) 으로 정규화 → +DI, -DI
  -DM (하락 방향 움직임) ─┘
  ADX = |+DI - -DI| / (+DI + -DI) 의 이동평균
```

| ADX 값 | 시장 상태 | 권장 전략 유형 |
|--------|---------|-------------|
| < 20 | 약한 추세·횡보 | 평균 회귀형 (볼린저 밴드, RSI) |
| 20 ~ 25 | 과도기 | 관망 권장 |
| 25 ~ 40 | 중간 추세 | 추세 추종형 (골든크로스, MACD) |
| > 40 | 강한 추세 | 추세 추종형, 청산 시점 주의 |

**ADX 활용 핵심**: 방향 전략의 게이트웨이 역할  
→ 추세장에서는 추세 전략을, 횡보장에서는 역추세 전략을 자동 선택

---

#### OBV — 거래량 균형 지표 (On-Balance Volume)

**원리**: 가격이 오른 날의 거래량은 더하고, 내린 날의 거래량은 빼서 누적합니다.  
가격보다 **거래량이 먼저 방향을 신호**하는 선행 지표입니다.

```
OBV 계산:
  종가 > 전일 종가 → OBV += 당일 거래량  (매수 세력 우위)
  종가 < 전일 종가 → OBV -= 당일 거래량  (매도 세력 우위)
  종가 = 전일 종가 → OBV 유지
```

| OBV 패턴 | 가격 동향 | 해석 | 신호 |
|---------|---------|------|------|
| OBV 상승 | 가격 상승 | 거래량이 추세 확인 | 추세 신뢰 ↑ |
| OBV 상승 | 가격 하락 | 하락 중 매집 발생 | 반등 예고 (매수) |
| OBV 하락 | 가격 상승 | 거래량 뒷받침 없음 | 상승 신뢰 ↓ (매도 주의) |
| OBV 하락 | 가격 하락 | 거래량이 하락 확인 | 추세 신뢰 ↑ |

**OBV 활용 핵심**: 가격 신호의 진위 검증  
→ 가격·OBV가 같은 방향 → 신호 신뢰도 높음  
→ 가격·OBV가 반대 방향(다이버전스) → 추세 전환 경고

## 전략 조합 분석

### 조합 1 · 골든크로스 + ADX (추세 강도 필터)

```
진입: 골든크로스 발생 AND ADX > 25
청산: 데드크로스 발생
```

| 항목 | 내용 |
|------|------|
| **목적** | 약한 추세에서의 휩쏘 차단 |
| **장점** | 횡보장 거짓 신호 대폭 감소, 진입 정확도 향상 |
| **단점** | 추세 초입(ADX 아직 낮음)에서 진입 기회 놓침, 거래 횟수 감소 |
| **적합 시장** | 방향성 있는 추세장 |
| **주의** | ADX 임계값 25는 고정값이 아님 — 자산·기간별 최적화 필요 |

---

### 조합 2 · 볼린저 밴드 + ADX (횡보 확인 필터)

```
진입: 볼린저 하단 돌파 AND ADX < 20
청산: 볼린저 상단 돌파 OR ADX > 25 (추세 전환 감지)
```

| 항목 | 내용 |
|------|------|
| **목적** | 추세장 역진입 방지 |
| **장점** | 강한 추세 중 역방향 진입 차단, 볼린저 밴드 성과 크게 개선 |
| **단점** | 신호 발생 빈도 감소, 박스권 탈출 직후 진입 기회 일부 손실 |
| **적합 시장** | 변동성 낮은 횡보·박스권 |
| **주의** | ADX < 20 구간이 길지 않으면 거래 횟수가 지나치게 적어질 수 있음 |

---

### 조합 3 · 골든크로스 + OBV (거래량 확인 필터)

```
진입: 골든크로스 발생 AND OBV가 상승 추세(OBV > OBV MA)
청산: 데드크로스 발생 OR OBV 다이버전스 감지
```

| 항목 | 내용 |
|------|------|
| **목적** | 거래량 뒷받침 없는 허위 골든크로스 차단 |
| **장점** | 세력 매수가 확인된 신호만 진입, 신뢰도 높은 추세 포착 |
| **단점** | OBV 이동평균 기간 선택 민감, 저유동성 종목에서 OBV 노이즈 심함 |
| **적합 시장** | 거래량 데이터가 풍부한 대형주·ETF |
| **주의** | OBV 절대값보다 추세 방향(기울기)이 중요 |

---

### 조합 4 · RSI + OBV (역추세 + 거래량 이중 확인)

```
진입: RSI < 30(과매도) AND OBV 상승 다이버전스(가격 하락 중 OBV 상승)
청산: RSI > 70(과매수) OR OBV 하락 전환
```

| 항목 | 내용 |
|------|------|
| **목적** | 과매도 구간에서 실제 매집 여부 확인 |
| **장점** | 하락 중 세력 매집 구간 정밀 포착, 낮은 리스크 진입 |
| **단점** | 다이버전스 감지 로직 구현 복잡, 신호 발생 빈도 매우 낮음 |
| **적합 시장** | 변동성 큰 조정 구간, 섹터 순환 전환점 |
| **주의** | 신호가 너무 희소해 충분한 샘플 수 확보가 어려울 수 있음 |

---

### 조합 5 · MACD + OBV (모멘텀 + 거래량 이중 확인)

```
진입: MACD 골든크로스(MACD > Signal) AND OBV 상승 추세
청산: MACD 데드크로스 OR OBV 급감(거래량 소멸)
```

| 항목 | 내용 |
|------|------|
| **목적** | 모멘텀과 거래량이 동시에 확인된 강한 신호만 선별 |
| **장점** | MACD 단독 대비 거짓 교차 신호 감소, 추세 신뢰도 향상 |
| **단점** | MACD가 원래 신호 횟수가 많아 OBV 필터 후에도 충분한 거래 가능 |
| **적합 시장** | 추세 전환이 명확한 구간 |
| **주의** | OBV 급감 청산 기준의 임계값 설정이 관건 |

---

### 조합 6 · ADX 국면 전환 스위처 (골든크로스 ↔ 볼린저 밴드)

```
ADX > 25 → 골든크로스 전략 활성화 (추세장 모드)
ADX < 20 → 볼린저 밴드 전략 활성화 (횡보장 모드)
20~25    → 기존 포지션 유지, 신규 진입 없음
```

| 항목 | 내용 |
|------|------|
| **목적** | 시장 국면에 따라 최적 전략 자동 전환 |
| **장점** | 단일 전략 대비 전천후 대응 가능, 각 전략의 약점 구간 회피 |
| **단점** | ADX 임계값 구간(20~25)에서 기회 손실, 국면 전환 지연 발생 |
| **적합 시장** | 추세장과 횡보장이 반복되는 중장기 데이터 |
| **주의** | 국면 전환 시 기존 포지션 처리 규칙을 명확히 정의해야 함 |

---

### 조합 7 · 골든크로스 + ADX + OBV (삼중 필터)

```
진입: 골든크로스 AND ADX > 25 AND OBV 상승 추세
청산: 데드크로스 OR ADX < 20 OR OBV 다이버전스
```

| 항목 | 내용 |
|------|------|
| **목적** | 방향(MA) + 강도(ADX) + 거래량(OBV) 세 조건 모두 충족 시만 진입 |
| **장점** | 신호 품질 최고 수준, 손실 거래 비율 대폭 감소 |
| **단점** | 신호 발생 빈도 매우 낮음, 큰 추세 초반 진입 기회 놓칠 가능성 |
| **적합 시장** | 장기 투자, 큰 움직임만 선택적으로 포착하는 전략 |
| **주의** | 과도한 필터링은 수익 기회 자체를 소멸시킬 수 있음 (과최적화 위험) |

## 전략 조합 선택 가이드

```
나의 목표는?
│
├─ 큰 추세를 타고 싶다
│   ├─ 신호 빈도 중요 → 골든크로스 + ADX
│   └─ 신호 품질 중요 → 골든크로스 + ADX + OBV  (조합 7)
│
├─ 박스권 매매를 하고 싶다
│   ├─ 단순하게 → 볼린저 밴드 + ADX  (조합 2)
│   └─ 바닥 정밀 포착 → RSI + OBV  (조합 4)
│
├─ 추세·횡보 상관없이 대응하고 싶다
│   └─ ADX 국면 전환 스위처  (조합 6)
│
└─ 거래량 확인이 최우선이다
    ├─ 추세장 → 골든크로스 + OBV  (조합 3)
    └─ 모멘텀 → MACD + OBV  (조합 5)
```
