# 인버스 ETF 매매 원칙

> 관련: [[적극투자형_전략]]

## 인버스 ETF 핵심 특성

### 1. 일별 복리 손실
- 인버스 ETF는 "일별 -1배"를 목표로 설계됨
- 장기 보유 시 실제 지수 하락분보다 수익이 적어짐
- 횡보장 장기 보유 시 손실 발생 가능

### 2. 변동성 드래그
```
예시: 지수가 -10% → +10% 반등 시
실제 지수:   100 → 90 → 99 (1% 손실)
인버스 ETF:  100 → 110 → 99 (1% 손실) ← 같이 손실
```
→ 횡보하는 하락장에서 특히 위험

## 매매 원칙

### 진입 조건 (까다롭게)

기본 조건:
- DOWNTREND 진입 (MA20 < MA60 < MA120 AND ADX > 25)

추가 조건 (모두 충족 시 진입):
- ADX > 30 (기존 25보다 높게 → 강한 추세만)
- 직전 5일 연속 하락 확인
- 거래량 증가 동반

### 청산 조건 (빠르게, 우선순위 순)

| 순위 | 조건 | 이유 |
|------|------|------|
| 1순위 | ADX 고점 대비 -5 하락 | 추세 약화 신호 |
| 2순위 | MA10이 MA20 상향 돌파 | 반등 초기 신호 |
| 3순위 | 진입 후 20거래일 경과 | 복리 손실 방지용 강제 청산 |

### 비중 제한

```
일반 UPTREND 매수: 최대 100%
인버스 ETF:        최대 30~50% (절반 이하)
```

인버스 ETF 자체의 구조적 리스크를 감안해 비중을 제한한다.

## 구현 순서 (권장)

```
1단계: DOWNTREND 현금 전략으로 백테스트 성과 확인
2단계: 인버스 ETF 전략 별도 백테스트
3단계: 두 전략 성과 비교
       → 인버스 ETF의 MDD가 낮고 수익이 높을 때만 실전 적용
```

## 주의사항

- 보유 기간: 짧을수록 좋음 (1~4주 이내 이상적)
- DOWNTREND가 수개월 지속되면 복리 손실로 실제 수익이 하락폭보다 작아짐
- 반등 신호 포착 즉시 청산 (반등 시 손실이 빠름)
