# 분할 매수/매도 원칙

> 관련 전략: [[투자성향/위험중립형_전략]] · [[투자성향/적극투자형_전략]]

---

## 왜 분할 매수/매도인가

한 번에 전액 투자하면 타이밍 실수 시 손실이 크다.
단계적으로 나눠서 진입·청산하면 **평균 단가를 개선**하고 **수익을 단계적으로 보호**할 수 있다.

```
일괄 매수: 진입 타이밍 하나에 모든 것이 달림 → 리스크 집중
분할 매수: 여러 조건에서 단계적 진입 → 평균 단가 안정화
```

---

## 분할 매수 3단계

| 단계 | 조건 | 목표 비중 | 의미 |
|------|------|----------|------|
| 1차 | UPTREND + 골든크로스(MA20 > MA60 돌파) | 40% | 추세 확인 후 초기 진입 |
| 2차 | UPTREND + MA20 지지 재확인 (60일 이내) | 70% | 추세 지속 확인 후 추가 매수 |
| 횡보 | SIDEWAYS + 볼린저밴드 하단 터치 | 30% | 횡보장 단기 반등 포착 |

**2차 매수 조건 상세:**
```
저가가 MA20 이하 터치 후 종가가 MA20 위로 마감
→ MA20이 지지선 역할을 확인한 것
→ 1차 매수 후 60거래일 이내에만 유효
```

---

## 분할 매도 3단계

| 단계 | 조건 | 유지 비중 | 의미 |
|------|------|----------|------|
| 1차 익절 | UPTREND → TRANSITION 전환 첫날 | 40% | 추세 약화 초기 신호에서 부분 익절 |
| 2차 청산 | 데드크로스 (MA20 < MA60 돌파) | 10% | 추세 전환 확인 후 추가 청산 |
| 전량 청산 | DOWNTREND 진입 | 0% | 하락 확정 시 전량 청산 (최우선) |
| 횡보 청산 | SIDEWAYS + 볼린저밴드 상단 터치 | 0% | 횡보 목표 달성 시 청산 |

**청산 우선순위:**
```
DOWNTREND > 데드크로스 > TRANSITION 전환 > 볼린저밴드 상단
```

## 포지션 크기 관리 방식

`targetpercent` 방식을 사용한다. 목표 비중을 지정하면 현재 보유량과의 차이만큼 자동 매매한다.

```python
size_series = NaN   # 기존 포지션 유지
size_series = 0.0   # 전량 청산
size_series = 0.4   # 40%로 조정
size_series = 0.7   # 70%로 조정
```

## 장점 / 단점

| | 내용 |
|---|------|
| **장점** | 단계적 진입으로 평균 단가 개선 |
| **장점** | TRANSITION 초기 신호에서 일부 먼저 익절 → 고점 수익 일부 보호 |
| **장점** | MA/볼린저밴드만 사용 → 신호가 단순하고 명확 |
| **단점** | 단계별 청산마다 수수료 발생 (3번 청산 = 3번 수수료) |
| **단점** | 10% 잔량은 금액이 너무 작아 실전에서 의미 없을 수 있음 |
| **단점** | DOWNTREND 판별이 늦으면 고점 대비 -10~15% 손실 후 청산 |

## 수수료 설정

```python
FEES     = 0.0015   # 0.15% (한국 주식 기준)
SLIPPAGE = 0.001    # 0.10%
```

분할 매도가 3단계이므로 수수료가 3회 발생한다.
10% 잔량 청산은 수익보다 수수료가 더 클 수 있으므로 실전 적용 시 검토 필요하다.
