# 적극투자형 전략

> 관련: [[투자성향_분류]] | 베이스: [[위험중립형_전략]]
> 매매원칙: [[매매원칙/분할매수매도_원칙]] · [[매매원칙/인버스ETF_매매원칙]]

## 성향 요약

위험중립형을 베이스로 **딱 2가지만 추가/변경**한다.
- 상승장: MA10 트리거로 더 빠른 진입 → 수익 극대화
- 하락장: 현금 대기 → 인버스 ETF로 수익화

---

## 위험중립형에서 변경한 지표와 이유

### [추가] MA10 진입 트리거 → [[TA지표/추세/MA_이동평균]]

**왜 MA10을 추가했는가?**
위험중립형의 골든크로스(MA20 > MA60)는 추세가 어느 정도 진행된 후에 발생한다.
MA10을 추가하면 추세 초입에 더 빠르게 진입할 수 있다.

**왜 국면 판별이 아닌 트리거로만 쓰는가?**
MA10을 국면 판별에 넣으면(MA10 > MA20 > MA60 > MA120) 4중 정렬 조건이 된다.
4중 정렬은 3중 정렬보다 충족 조건이 더 까다로워 오히려 UPTREND 신호가 줄어든다.
MA10은 국면 확인 후 **진입 타이밍에만** 쓰는 것이 의도한 효과를 낸다.

**MA10/60/120이나 MA10/20/60/120 전체 정렬을 쓰지 않는 이유**
- MA10/60/120: MA10과 MA60 사이 간격(50일)이 너무 커 휩소 급증
- MA10/20/60/120 전체 정렬: 4중 조건으로 신호 감소 → 더 보수적으로 작동

### [변경] DOWNTREND: 현금 대기 → 인버스 ETF

**왜 현금 대신 인버스 ETF인가?**
위험중립형은 DOWNTREND에서 수익이 0이다.
인버스 ETF는 하락장을 수익 기회로 전환한다.
하락장을 "회피"에서 "공격"으로 바꾸는 것이 위험중립형 → 적극투자형의 핵심 변화다.

**왜 비중을 50% 이하로 제한하는가?**
인버스 ETF는 일별 복리 구조로 장기 보유 시 실제 하락분보다 수익이 적어진다.
또한 반등 시 손실이 빠르게 발생하므로 일반 매수보다 보수적 비중을 유지한다.

→ 상세 원칙: [[매매원칙/인버스ETF_매매원칙]]

## 국면별 지표 조합 방식

### UPTREND (변경)
```
[기존] MA20/60/120 정렬 + ADX > 25 → 골든크로스 발생 시 매수
[변경] MA20/60/120 정렬 + ADX > 25 → MA10이 MA20 상향 돌파 시 매수 (더 빠른 진입)
```

### SIDEWAYS (변경 없음)
```
ADX < 20 → 볼린저밴드 하단 터치 시 매수 (위험중립형과 동일)
```

### TRANSITION (변경 없음)
```
신규 진입 차단, 기존 포지션 유지 (위험중립형과 동일)
```

### DOWNTREND (변경)
```
[기존] 전량 현금 대기
[변경] ADX > 30 + 직전 5일 연속 하락 + 거래량 증가 → 인버스 ETF 매수 (최대 50%)
```

## 위험중립형과의 차이 요약

| 국면 | 위험중립형 | 적극투자형 |
|------|-----------|-----------|
| UPTREND 진입 | 골든크로스 (MA20>MA60) | **MA10이 MA20 상향 돌파** (더 빠름) |
| SIDEWAYS | 볼린저밴드 50% | 동일 |
| TRANSITION | 포지션 유지 | 동일 |
| DOWNTREND | 현금 0% | **인버스 ETF 최대 50%** |

---

## 변경 없는 항목 (위험중립형 그대로)

- 4국면 판별 로직 (MA20/60/120 + ADX)
- SIDEWAYS 볼린저밴드 전략
- Walk-Forward 최적화
- 분할 매수/매도 구조 → [[매매원칙/분할매수매도_원칙]]
- 멀티종목 포트폴리오 구성 + 모멘텀 비중 배분

## 성과 평가 지표

목표: **연 CAGR 10% 이상, 인버스 ETF로 하락장도 수익화**

| 구분 | 지표 | 기준 | 의미 |
|------|------|------|------|
| 1차 관문 | **CAGR** | 10% 이상 | 미달 시 위험중립형보다 복잡하기만 하고 이점 없음 |
| 핵심 | **샤프비율** | 1.0 이상 | 인버스 ETF 복잡성이 변동성 통제와 수익 향상 모두에 기여하는지 확인 |
| 한계 | **MDD** | -25% 이내 | 인버스 ETF 타이밍 실패 시 손실 한계 모니터링 |
| 한계 | **MDD Duration** | 12개월 이내 | 초과 시 인버스 ETF 전략 재검토 |
| 신호 품질 | **Profit Factor** | 1.5 이상 | 인버스 ETF 신호 유효성. 승률 대신 사용 |

→ 각 지표 상세: [[성과지표/CAGR]] · [[성과지표/샤프비율]] · [[성과지표/MDD]] · [[성과지표/MDD_Duration]] · [[성과지표/Profit_Factor]]

**판단 흐름**
```
CAGR 10% 미만        → 전략 실패 (인버스 ETF 추가 의미 없음)
CAGR 10% 이상        → 샤프비율 1.0 이상 확인
  → 통과 시 MDD -25% 이내 + Duration 12개월 이내 확인
  → 통과 시 Profit Factor 1.5 이상이면 성공
```

**Walk-Forward 최적화 metric:** `sharpe_ratio`
