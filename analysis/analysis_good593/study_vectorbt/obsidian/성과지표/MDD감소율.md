# MDD감소율 · MDD 개선율 (vs KOSPI)

> 분류: 성과지표 | 관련: [[성과지표/MDD]] · [[성과지표/Calmar비율]] · [[성과지표/Beta]]

---

## 개념

전략의 MDD가 KOSPI 대비 **얼마나 개선됐는지**를 비율로 나타낸다.
하락 방어 전략이 실제로 얼마나 손실을 줄였는지 직접 측정한다.

```
MDD감소율 = (전략 MDD − KOSPI MDD) / |KOSPI MDD|

전략 MDD -20%, KOSPI MDD -40% → (−0.20 − (−0.40)) / 0.40 = +50%  (50% 개선)
전략 MDD -40%, KOSPI MDD -40% →  0%                                 (동일)
전략 MDD -50%, KOSPI MDD -40% → (−0.50 − (−0.40)) / 0.40 = −25%  (25% 악화)
```

---

## 계산 방법

```python
# stock_system/backtest/metrics/calc.py
eq_mdd        = (eq / eq.cummax() - 1).min()   # 정규화된 전략 MDD
bm_mdd        = (bm / bm.cummax() - 1).min()   # 정규화된 벤치마크 MDD
mdd_reduction = (eq_mdd - bm_mdd) / abs(bm_mdd) if abs(bm_mdd) > 1e-12 else np.nan
```

양수 = 전략이 KOSPI보다 MDD가 작음 (개선)
음수 = 전략이 KOSPI보다 MDD가 큼 (악화)

---

## 해석 기준

| MDD감소율 | 의미 |
|----------|------|
| > 30% | 우수한 하락 방어 |
| 10 ~ 30% | 유의미한 개선 |
| 0 ~ 10% | 소폭 개선 — 효과 미미 |
| < 0% | KOSPI보다 더 많이 떨어짐 → 하락 방어 실패 |

---

## 장점 / 단점

| | 내용 |
|---|---|
| **장점** | 하락 방어 효과를 단일 숫자로 직관적으로 표현 |
| **단점** | 벤치마크 MDD가 작은 구간에서는 값이 불안정 |
| **단단점** | 기간에 따라 KOSPI MDD 자체가 크게 달라짐 (분모 변동) |

---

## 투자성향별 적용

| 성향 | 기준 | 비고 |
|------|------|------|
| **위험중립형** | **20% 목표 / 10% 경보** | 하락 방어가 핵심 목적이므로 엄격하게 적용 |
| **적극투자형** | **0% 목표 / −10% 경보** | 최소한 KOSPI 수준의 손실을 넘지 않아야 함 |
