# Alpha · 초과 수익률

> 분류: 성과지표 | 관련: [[성과지표/CAGR]] · [[성과지표/Beta]] · [[성과지표/IR]]

---

## 개념

벤치마크(KOSPI) 대비 전략이 **추가로 달성한 연간 수익률**이다.
시장 자체의 수익이 아닌, 전략이 독자적으로 만들어낸 가치를 측정한다.

```
Alpha = 전략 CAGR − 벤치마크 CAGR

전략 CAGR 12%, KOSPI CAGR 8% → Alpha = +4%
전략 CAGR  6%, KOSPI CAGR 8% → Alpha = -2%  (인덱스 펀드만 못함)
```

---

## 계산 방법

```python
# stock_system/backtest/metrics/calc.py
n_y     = max(len(eq) / 252, 1e-6)
eq_cagr = eq.iloc[-1] ** (1 / n_y) - 1   # eq: 정규화된 전략 자산곡선
bm_cagr = bm.iloc[-1] ** (1 / n_y) - 1   # bm: 정규화된 벤치마크
alpha   = eq_cagr - bm_cagr
```

---

## 해석 기준

| Alpha | 의미 |
|-------|------|
| < 0% | 전략이 인덱스 펀드보다 못함 → 전략 가치 없음 |
| 0 ~ +2% | 소폭 초과 — 비용(세금·슬리피지) 감안하면 실질 효과 미미 |
| +2 ~ +5% | 유의미한 초과 수익 |
| > +5% | 우수한 초과 수익 |

Alpha < 0이면 나머지 지표를 볼 필요 없이 전략이 실패한 것으로 판단한다.

---

## 장점 / 단점

| | 내용 |
|---|---|
| **장점** | 시장 상승분을 제거해 전략 고유의 기여도를 명확히 분리 |
| **단점** | CAGR 단순 차이이므로 리스크 조정이 없음 → [[성과지표/IR]]과 함께 봐야 함 |
| **단점** | 벤치마크 선택에 따라 값이 크게 달라짐 |

---

## 투자성향별 적용

| 성향 | 기준 | 비고 |
|------|------|------|
| **위험중립형** | **+2% 목표 / 0% 경보** | 0% 이하면 KOSPI 인덱스 펀드보다 못함 → 전략 실패 |
| **적극투자형** | **+5% 목표 / +2% 경보** | 인버스 ETF까지 활용하는 복잡성이 정당화되려면 충분한 초과 수익 필요 |

## 이 시스템에서의 역할

`stock_system/backtest/metrics/calc.py::calc_metrics()`에서 KOSPI 대비 상대 지표로 산출된다.
벤치마크가 없으면 계산하지 않는다.
