# Calmar 비율 (Calmar Ratio)

> 분류: 성과지표 | 관련: [[성과지표/CAGR]] · [[성과지표/MDD]] · [[성과지표/샤프비율]]

---

## 개념

**MDD(최대 낙폭) 한 단위당 CAGR이 얼마인가**를 나타낸다.
하락 리스크 대비 수익 효율을 측정하는 데 특화된 지표다.

```
Calmar = CAGR / |MDD|

CAGR 15%, MDD -30% → Calmar = 0.5
CAGR 10%, MDD -10% → Calmar = 1.0
→ 두 번째 전략이 하락 리스크 대비 더 효율적
```

---

## 계산 방법

```python
cagr   = (equity.iloc[-1] / equity.iloc[0]) ** (1 / n_years) - 1
mdd    = (equity / equity.cummax() - 1).min()
calmar = cagr / abs(mdd) if mdd < 0 else np.nan
```

MDD가 0이면(손실이 전혀 없으면) 정의되지 않으므로 NaN 처리한다.

---

## 해석 기준

| Calmar | 의미 |
|--------|------|
| < 0.5 | 낮음 — 하락 리스크 대비 수익이 부족 |
| 0.5 ~ 1.0 | 보통 |
| 1.0 ~ 2.0 | 우수 |
| > 2.0 | 매우 우수 |

---

## 샤프비율과의 차이

| | 샤프비율 | Calmar비율 |
|---|---------|-----------|
| 리스크 정의 | 전체 변동성 (상승+하락) | 최대 낙폭 (하락만) |
| 적합한 상황 | 일반적인 전략 비교 | MDD 관리가 중요한 전략 |
| 단점 | 상승 변동성도 패널티 | 단 한 번의 MDD에 민감 |

위험중립형 전략처럼 **MDD 최소화가 목표**일 때 Calmar비율이 더 직접적인 평가 지표다.

---

## 장점 / 단점

| | 내용 |
|---|---|
| **장점** | 투자자가 실제로 경험하는 최악의 손실을 분모로 사용 |
| **장점** | 하락 방어 전략 평가에 샤프비율보다 적합 |
| **단점** | 단 한 번의 극단적 MDD로 값이 크게 낮아짐 |
| **단점** | 드로다운 기간(회복 속도)은 반영하지 않음 |

---

## 투자성향별 적용

| 성향 | 기준 | 역할 |
|------|------|------|
| **위험중립형** | **0.5 이상** | 핵심 효율 지표. 낮은 수익(5~10%) + 낮은 MDD 조합이 의도대로 작동하는지 확인. WF metric으로 사용 |
| 적극투자형 | 참고용 | MDD 최소화가 목표가 아니므로 보조 확인만. 샤프비율이 주 지표 |

위험중립형 기준: CAGR 7%, MDD -14% → Calmar 0.5 → 통과
위험중립형 기준: CAGR 7%, MDD -30% → Calmar 0.23 → 하락 방어 실패

---

## 이 시스템에서의 역할

`portfolio_backtest.py::_calc_metrics()`에서 성과 비교 테이블에 포함된다.
Walk-Forward 최적화의 `metric` 파라미터로 `calmar_ratio`를 선택할 수도 있다.

```python
# 위험중립형: calmar_ratio 권장
result = walk_forward(..., metric="calmar_ratio")

# 적극투자형: sharpe_ratio (기본값)
result = walk_forward(..., metric="sharpe_ratio")
```
