# IR · Information Ratio · 정보 비율

> 분류: 성과지표 | 관련: [[성과지표/Alpha]] · [[성과지표/샤프비율]] · [[성과지표/Sortino비율]]

---

## 개념

벤치마크(KOSPI) 대비 초과 수익을 **얼마나 일관되게** 달성했는지를 나타낸다.
Alpha의 크기뿐 아니라 **안정성(변동성)**까지 함께 평가한다.

```
IR = 평균 초과 수익률 / 초과 수익률의 표준편차 × √252

초과 수익률 = 전략 일별 수익률 − 벤치마크 일별 수익률
```

```
예시:
  A 전략: 매달 꾸준히 +1% 초과  → IR 높음 (안정적)
  B 전략: 어떤 달은 +5%, 어떤 달은 -3% 초과  → IR 낮음 (불안정)
  두 전략의 연간 Alpha는 같아도 IR은 크게 다를 수 있다
```

---

## 계산 방법

```python
# stock_system/backtest/metrics/calc.py
excess = eq_ret - bm_ret   # 일별 초과 수익률
ir = excess.mean() / excess.std() * np.sqrt(252) if excess.std() > 1e-12 else np.nan
```

---

## 해석 기준

| IR | 의미 |
|----|------|
| < 0 | 평균적으로 벤치마크를 하회 |
| 0 ~ 0.3 | 낮음 — 초과 수익이 불안정 |
| 0.3 ~ 0.5 | 보통 |
| 0.5 ~ 1.0 | 우수한 일관성 |
| > 1.0 | 매우 우수 (전문 액티브 펀드 기준) |

---

## 샤프비율과의 차이

| | 샤프비율 | IR |
|---|---------|-----|
| 기준점 | 무위험 수익률 (국채) | 벤치마크 (KOSPI) |
| 측정 대상 | 절대 수익의 효율 | 벤치마크 대비 초과 수익의 효율 |
| 용도 | 전략 자체의 리스크 효율 | 전략이 시장을 일관되게 이기는지 |

---

## 장점 / 단점

| | 내용 |
|---|---|
| **장점** | Alpha의 크기와 안정성을 동시에 평가 |
| **장점** | 운이 아닌 실력 기반의 초과 수익인지 판별 가능 |
| **단점** | 벤치마크 선택에 따라 값이 달라짐 |
| **단점** | 초과 수익이 음수인 구간이 많으면 해석이 어려움 |

## 투자성향별 적용

| 성향 | 기준 | 비고 |
|------|------|------|
| **위험중립형** | **0.2 목표 / 0.0 경보** | 낮은 베타 구조상 IR 자체가 높지 않아도 됨. 0 이상이면 일관된 초과 수익 확인 |
| **적극투자형** | **0.5 목표 / 0.2 경보** | 공격적 전략인 만큼 초과 수익이 크고 일관되어야 함 |

## 이 시스템에서의 역할

`stock_system/backtest/metrics/calc.py::calc_metrics()`에서 KOSPI 대비 상대 지표로 산출된다.
[[성과지표/Alpha]]가 높더라도 IR이 낮으면 단발성 수익일 가능성이 높다.
