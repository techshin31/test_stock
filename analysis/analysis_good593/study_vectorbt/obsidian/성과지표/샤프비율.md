# 샤프비율 (Sharpe Ratio)

> 분류: 성과지표 | 관련: [[성과지표/CAGR]] · [[성과지표/MDD]] · [[성과지표/Calmar비율]]
> 최적화 기준: [[최적화/Walk_Forward_최적화]] · [[최적화/Grid_Search_최적화]]

---

## 개념

**위험(변동성) 한 단위당 얼마나 수익을 올렸는가**를 나타낸다.
수익률이 같아도 변동성이 낮은 전략이 더 높은 샤프비율을 가진다.

```
전략 A: 연수익 20%, 연변동성 40% → Sharpe ≈ 0.5
전략 B: 연수익 10%, 연변동성 10% → Sharpe ≈ 1.0
→ B가 리스크 대비 효율적
```

---

## 계산 방법

```python
dr     = equity.pct_change().dropna()          # 일별 수익률
sharpe = dr.mean() / dr.std() * np.sqrt(252)   # 연환산
```

`sqrt(252)`: 연간 거래일 수로 환산 (일→연)
무위험 수익률(risk-free rate)은 단순화를 위해 0으로 가정한다.

---

## 해석 기준

| 샤프비율 | 의미 |
|----------|------|
| < 0 | 무위험 자산보다 못함 |
| 0 ~ 0.5 | 미흡 |
| 0.5 ~ 1.0 | 보통 |
| 1.0 ~ 2.0 | 우수 |
| > 2.0 | 매우 우수 (지속 가능성 의심해볼 필요 있음) |

---

## 장점 / 단점

| | 내용 |
|---|---|
| **장점** | 수익과 리스크를 동시에 반영하는 가장 널리 쓰이는 지표 |
| **장점** | 전략 간 비교에 적합 — 수익률만 비교하는 것보다 공정 |
| **단점** | 변동성을 리스크로 정의 → 상승 변동성도 패널티로 처리 |
| **단점** | 정규분포 가정 → 극단적 손실(팻테일) 리스크를 과소평가 |
| **단점** | 무위험 수익률 설정에 따라 달라짐 |

---

## 투자성향별 적용

| 성향 | 기준 | 역할 |
|------|------|------|
| 위험중립형 | 참고용 | 하락 방어 목표에 부적합 — [[성과지표/Sortino비율]]로 대체 권장 |
| **적극투자형** | **1.0 이상** | 핵심 지표. 인버스 ETF 복잡성이 변동성 통제와 수익 향상 모두에 기여하는지 확인 |

위험중립형에서 샤프비율이 낮게 나와도 Sortino비율이 높으면 전략이 의도대로 작동하는 것이다.
상승 변동성(좋은 변동성)을 리스크로 패널티 처리하기 때문이다.

## 이 시스템에서의 역할

**Walk-Forward 최적화의 기본 metric**으로 사용된다.

```python
# optimizer.py::walk_forward()
# 위험중립형: calmar_ratio 권장
# 적극투자형: sharpe_ratio (기본값 유지)
metric = "sharpe_ratio"
```

Grid Search 결과도 샤프비율 내림차순으로 정렬해 상위 파라미터를 우선 확인한다.
