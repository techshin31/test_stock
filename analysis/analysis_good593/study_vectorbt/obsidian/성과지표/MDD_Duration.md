# MDD Duration · 최대 낙폭 지속 기간

> 분류: 성과지표 | 관련: [[성과지표/MDD]] · [[성과지표/Calmar비율]] · [[성과지표/CAGR]]

---

## 개념

**고점에서 저점까지 하락 후 고점을 회복하기까지 걸린 최장 기간**이다.
MDD가 "얼마나 떨어졌는가"라면, MDD Duration은 "얼마나 오래 손실 상태였는가"다.

```
고점(100) → 하락 → 저점(80) → 회복 → 고점(100)
                 ←────── Duration ──────→
```

자동매매에서 특히 중요하다.
MDD -20%가 1개월 안에 회복되면 버틸 수 있지만,
MDD -20%가 2년 지속되면 실전에서 시스템을 끄게 된다.

---

## 계산 방법

```python
# 드로다운 시작 ~ 고점 회복까지의 기간
drawdown = equity / equity.cummax() - 1
in_drawdown = drawdown < 0

# 연속 손실 구간의 최대 길이 (거래일 기준)
max_duration = 0
current = 0
for d in in_drawdown:
    current = current + 1 if d else 0
    max_duration = max(max_duration, current)
```

---

## 해석 기준

| Duration | 의미 |
|----------|------|
| 1개월 이내 | 매우 양호 |
| 1~3개월 | 양호 |
| 3~6개월 | 보통 — 심리적 압박 시작 |
| 6~12개월 | 위험 — 운용자 개입 가능성 |
| 12개월 이상 | 전략 재검토 필요 |

---

## MDD와 함께 봐야 하는 이유

MDD만으로는 실전 운용 가능성을 판단할 수 없다.

| MDD | Duration | 판단 |
|-----|----------|------|
| -20% | 1개월 | 감내 가능 — 빠른 회복 |
| -10% | 18개월 | 위험 — 장기 손실 상태가 더 치명적 |
| -30% | 2개월 | 심리적 고통 크지만 자동매매 유지 가능 |
| -15% | 24개월 | 시스템 신뢰 붕괴 → 실전 운용 불가 |

---

## 장점 / 단점

| | 내용 |
|---|---|
| **장점** | 자동매매 실전 운용 가능성을 판단하는 가장 현실적인 지표 |
| **장점** | MDD와 함께 리스크의 깊이와 기간을 모두 파악 가능 |
| **단점** | 백테스트 기간이 짧으면 최악의 Duration이 포함되지 않을 수 있음 |
| **단점** | 계산 기준(거래일/달력일)에 따라 값이 달라짐 |

## 투자성향별 적용

| 성향 | 기준 | 이유 |
|------|------|------|
| **위험중립형** | **6개월 이내** | 낮은 수익 목표(5~10%)인 만큼 회복도 빨라야 함. 6개월 초과 시 전략 가치 의문 |
| **적극투자형** | **12개월 이내** | 인버스 ETF로 하락장 수익화를 시도하므로 더 긴 Duration 허용 가능. 단 12개월 초과 시 재검토 |
