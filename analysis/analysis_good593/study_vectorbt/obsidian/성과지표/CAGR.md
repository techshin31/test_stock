# CAGR · 연평균 복리 수익률 (Compound Annual Growth Rate)

> 분류: 성과지표 | 관련: [[성과지표/MDD]] · [[성과지표/Calmar비율]] · [[성과지표/샤프비율]]

---

## 개념

투자 기간 전체의 수익률을 **연 단위 복리**로 환산한 값이다.
총 수익률과 달리 기간이 달라도 전략 간 비교가 가능하다.

```
총 수익률 100%: 1년이면 대단하지만, 10년이면 평범하다
CAGR로 환산하면 → 1년: 100% / 10년: 7.2%  → 직접 비교 가능
```

---

## 계산 방법

```python
n_years = (equity.index[-1] - equity.index[0]).days / 365.25
cagr = (equity.iloc[-1] / equity.iloc[0]) ** (1 / n_years) - 1
```

---

## 해석 기준

| CAGR | 의미 |
|------|------|
| < 0% | 손실 구간 |
| 0~5% | 예금 수준 |
| 5~10% | 시장 평균 수준 (코스피 장기 평균) |
| 10~20% | 우수한 전략 |
| > 20% | 매우 높음 — 리스크 동반 가능성 확인 필요 |

CAGR 단독으로는 리스크를 알 수 없다.
반드시 [[성과지표/MDD]]와 함께 봐야 한다 → [[성과지표/Calmar비율]]

## 장점 / 단점

| | 내용 |
|---|---|
| **장점** | 기간이 다른 전략도 동일 기준으로 비교 가능 |
| **장점** | 복리 효과를 반영해 실제 자산 성장률을 직관적으로 표현 |
| **단점** | 중간 구간의 변동성(MDD, 드로다운 기간)을 무시 |
| **단점** | 시작/종료 시점에 따라 크게 달라질 수 있음 |

---

## 투자성향별 적용

| 성향 | 기준 | 판단 기준 |
|------|------|----------|
| **위험중립형** | **5~10%** | 1차 관문. 5% 미만 → 예금과 다를 바 없음. 10% 초과 → MDD 점검 필수 (과도한 리스크 가능) |
| **적극투자형** | **10% 이상** | 1차 관문. 미달 시 인버스 ETF까지 쓰는 복잡성이 정당화되지 않음 |

두 성향 모두 CAGR이 1차 관문이다.
CAGR을 통과하지 못하면 나머지 지표를 볼 필요가 없다.

`portfolio_backtest.py::_calc_metrics()`에서 전략 비교 테이블의 핵심 지표로 사용된다.
Walk-Forward 최적화의 `metric` 파라미터로는 사용하지 않고, 최종 성과 보고에만 활용한다.
