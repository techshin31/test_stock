# Sortino 비율 (Sortino Ratio)

> 분류: 성과지표 | 관련: [[성과지표/샤프비율]] · [[성과지표/MDD]] · [[성과지표/Calmar비율]]

---

## 개념

샤프비율의 변형으로, **하락 변동성만 리스크로 처리**한다.
샤프비율은 상승 변동성도 패널티로 포함하지만, Sortino는 손실 가능성만 측정한다.

```
샤프비율: 수익률 / 전체 변동성  (상승+하락 모두 패널티)
Sortino:  수익률 / 하락 변동성  (하락만 패널티)
```

하락 방어가 목표인 전략에서 샤프비율보다 더 정확한 평가가 가능하다.

---

## 계산 방법

```python
dr = equity.pct_change().dropna()

# 하락 변동성 = 음수 수익률만의 표준편차
downside_returns = dr[dr < 0]
downside_std = downside_returns.std() * np.sqrt(252)

sortino = dr.mean() * 252 / downside_std if downside_std > 0 else np.nan
```

---

## 샤프비율과의 차이

| | 샤프비율 | Sortino비율 |
|---|---------|------------|
| 리스크 정의 | 전체 변동성 (상승+하락) | 하락 변동성만 |
| 상승 변동성 | 패널티 포함 | 패널티 없음 |
| 적합한 전략 | 일반적인 비교 | 하락 방어 목표 전략 |
| 값 크기 | 상대적으로 낮음 | 상대적으로 높음 |

같은 전략이라도 Sortino가 샤프비율보다 항상 높거나 같다.
두 값 차이가 클수록 상승 변동성이 많다는 의미다.

---

## 해석 기준

| Sortino | 의미 |
|---------|------|
| < 1.0 | 미흡 |
| 1.0 ~ 2.0 | 양호 |
| 2.0 이상 | 우수 |

샤프비율 기준보다 0.5~1.0 높게 나오는 게 일반적이다.

---

## 장점 / 단점

| | 내용 |
|---|---|
| **장점** | 하락 방어 전략의 실질 성과를 샤프비율보다 정확하게 반영 |
| **장점** | 좋은 변동성(상승)을 리스크로 처리하지 않아 전략을 과소평가하지 않음 |
| **단점** | 하락 횟수가 적으면 하락 변동성 계산이 불안정 |
| **단점** | 샤프비율만큼 업계에서 보편화되지 않아 타 전략과 비교가 어려울 수 있음 |

---

## 투자성향별 적용

| 성향 | 기준 | 역할 |
|------|------|------|
| **위험중립형** | **1.0 이상** | **샤프비율 대체 — 하락 방어 목표에 더 적합한 효율 지표** |
| 적극투자형 | 참고용 | 샤프비율이 주 지표. Sortino는 보조로 확인 |

위험중립형은 DOWNTREND 전량 현금 대기로 하락 변동성 자체를 줄이는 전략이므로,
Sortino비율이 샤프비율보다 실제 전략 성과를 더 잘 반영한다.
