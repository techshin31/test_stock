# Beta · 시장 민감도

> 분류: 성과지표 | 관련: [[성과지표/Alpha]] · [[성과지표/IR]] · [[성과지표/MDD]]

---

## 개념

전략 수익률이 벤치마크(KOSPI) 수익률에 얼마나 동조하는지를 나타낸다.
KOSPI가 1% 움직일 때 전략이 몇 % 움직이는지를 측정한다.

```
Beta = 1.0 → 전략이 KOSPI와 동일하게 움직임
Beta = 0.6 → KOSPI가 -10%일 때 전략은 -6% (하락 방어)
Beta = 1.4 → KOSPI가 -10%일 때 전략은 -14% (시장보다 더 크게 하락)
```

---

## 계산 방법

```python
# stock_system/backtest/metrics/calc.py
cov  = np.cov(eq_ret.values, bm_ret.values)  # 일별 수익률 공분산 행렬
beta = cov[0, 1] / cov[1, 1]                 # 전략-시장 공분산 / 시장 분산
```

---

## 해석 기준

| Beta | 의미 |
|------|------|
| < 0 | 시장과 역방향 (인버스 전략) |
| 0 ~ 0.5 | 시장 변동에 매우 둔감 |
| 0.5 ~ 0.8 | 하락 방어 효과 있음 |
| 0.8 ~ 1.0 | 시장과 유사하게 움직임 |
| 1.0 ~ 1.3 | 시장보다 변동이 큰 공격적 전략 |
| > 1.5 | 과도한 시장 노출 — 리스크 점검 필요 |

Beta가 낮다고 무조건 좋은 것은 아니다.
Beta가 낮으면서 [[성과지표/Alpha]]가 높아야 의미 있는 하락 방어다.

## 장점 / 단점

| | 내용 |
|---|---|
| **장점** | 전략의 시장 노출 수준을 단일 숫자로 표현 |
| **단점** | 과거 데이터 기반 — 시장 국면 전환 시 Beta 자체가 달라질 수 있음 |
| **단점** | 비선형 관계(하락장 Beta ≠ 상승장 Beta)를 하나의 값으로 압축 |

---

## 투자성향별 적용

| 성향 | 기준 | 비고 |
|------|------|------|
| **위험중립형** | **≤ 0.8 목표 / ≤ 1.0 경보** | 하락 방어가 핵심 — Beta가 1을 넘으면 위험중립 목적 달성 실패 |
| **적극투자형** | **≤ 1.3 목표 / ≤ 1.5 경보** | 공격적 전략이므로 Beta가 높아도 되지만 무한정 허용하지 않음 |

## 이 시스템에서의 역할

`stock_system/backtest/metrics/calc.py::calc_metrics()`에서 KOSPI 대비 상대 지표로 산출된다.
위험중립형에서 하락 방어 효과를 수치로 확인하는 데 사용된다.
