# MDD · 최대 낙폭 (Maximum Drawdown)

> 분류: 성과지표 | 관련: [[성과지표/CAGR]] · [[성과지표/Calmar비율]] · [[성과지표/샤프비율]]

---

## 개념

전략 운용 기간 중 **고점 대비 최대 손실폭**이다.
"가장 나쁜 타이밍에 투자했을 때 얼마나 잃을 수 있는가"를 나타낸다.

```
자산가치: 100 → 150 → 90 → 130
고점: 150, 저점: 90
MDD = (90 - 150) / 150 = -40%
```

음수로 표현되며, 0에 가까울수록 좋다.

---

## 계산 방법

```python
mdd = (equity / equity.cummax() - 1).min()
```

`equity.cummax()`: 현재까지의 최고 자산가치
`equity / equity.cummax() - 1`: 고점 대비 현재 낙폭 (드로다운)
`.min()`: 드로다운 중 가장 낮은 값 = MDD

---

## 해석 기준

| MDD | 의미 |
|-----|------|
| 0 ~ -10% | 매우 안정적 |
| -10 ~ -20% | 양호 |
| -20 ~ -30% | 보통 (장기 투자 시 버틸 수 있는 수준) |
| -30 ~ -50% | 위험 — 심리적으로 버티기 어려움 |
| -50% 이하 | 극도로 위험 |

위험중립형 전략의 목표는 **MDD 최소화**다. → [[투자성향/위험중립형_전략]]

## 장점 / 단점

| | 내용 |
|---|---|
| **장점** | 최악의 시나리오를 직관적으로 표현 |
| **장점** | 실제 투자자가 경험하는 심리적 고통과 가장 밀접 |
| **단점** | 드로다운 기간(얼마나 오래 손실 상태였는가)은 알 수 없음 |
| **단점** | 단 한 번의 극단적 사건에도 크게 영향받음 |

---

## 투자성향별 적용

| 성향 | 기준 | 역할 |
|------|------|------|
| **위험중립형** | **-15% 이내** | 핵심 지표. 전략의 존재 이유가 하락 방어이므로 이 기준을 넘기면 전략 실패 |
| **적극투자형** | **-25% 이내** | 한계 지표. 인버스 ETF로 더 공격적이므로 기준 완화. 단 초과 시 인버스 신호 재검토 |

MDD는 [[성과지표/MDD_Duration]]과 반드시 함께 봐야 한다.
MDD 수치가 같아도 회복 기간이 다르면 실전 운용 가능성이 완전히 달라진다.

## 이 시스템에서의 역할

**성과 비교 테이블**에서 전략의 리스크를 평가하는 핵심 지표로 사용된다.
[[성과지표/CAGR]]과 함께 [[성과지표/Calmar비율]]을 계산하는 데 쓰인다.

```python
# portfolio_backtest.py::_calc_metrics()
mdd = (equity / equity.cummax() - 1).min()
```

위험중립형 전략은 DOWNTREND 국면에서 전량 현금 대기를 통해 MDD를 제한한다.
