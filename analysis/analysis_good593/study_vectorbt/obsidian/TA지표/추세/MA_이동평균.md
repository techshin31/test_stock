# MA · 이동평균 (Moving Average)

> 분류: 추세 지표 | 관련 전략: [[투자성향/위험중립형_전략]] · [[투자성향/적극투자형_전략]]

---

## 개념

일정 기간 종가의 **평균값**을 연결한 선이다.
가격의 단기 노이즈를 제거해 **추세 방향을 시각적으로 파악**하는 데 쓴다.

```
MA20  = 최근 20일 종가의 평균   → 단기 추세
MA60  = 최근 60일 종가의 평균   → 중기 추세
MA120 = 최근 120일 종가의 평균  → 장기 추세
```

---

## 계산 방법

```python
ma20  = close.rolling(20).mean()
ma60  = close.rolling(60).mean()
ma120 = close.rolling(120).mean()
```

---

## 골든크로스 / 데드크로스

MA 두 개의 교차를 **진입·청산 트리거**로 사용한다.

| 신호 | 조건 | 의미 |
|------|------|------|
| 골든크로스 | 단기 MA가 장기 MA를 **상향** 돌파 | 상승 추세 시작 → 매수 |
| 데드크로스 | 단기 MA가 장기 MA를 **하향** 돌파 | 하락 추세 시작 → 매도 |

```python
# 골든크로스: 이전 봉 fast < slow, 현재 봉 fast >= slow
entries = (fast_ma >= slow_ma) & (fast_ma.shift(1) < slow_ma.shift(1))
exits   = (fast_ma <  slow_ma) & (fast_ma.shift(1) >= slow_ma.shift(1))
```

---

## 3중 MA 정렬

단일 MA 교차보다 **3개 MA가 모두 정렬**됐을 때 신호 신뢰도가 높아진다.

```
상승 정렬: MA20 > MA60 > MA120  → 단·중·장기 모두 상승 방향
하락 정렬: MA20 < MA60 < MA120  → 단·중·장기 모두 하락 방향
```

2개 정렬만으로는 단기 노이즈일 가능성이 있으므로 3중 구조를 사용한다.

---

## 장점 / 단점

| | 내용 |
|---|------|
| **장점** | 계산이 단순하고 직관적 |
| **장점** | 기간이 길수록 노이즈 제거 효과 증가 |
| **장점** | 3중 정렬 시 추세 방향 신뢰도 높음 |
| **단점** | 후행 지표 — 추세가 시작된 후에야 신호 발생 |
| **단점** | 횡보장에서 골든크로스/데드크로스가 반복되며 손실 발생 (휩소) |
| **단점** | MA 기간 선택에 따라 결과가 크게 달라짐 |

---

## 이 시스템에서의 역할

### 위험중립형 전략

| 용도 | 내용 |
|------|------|
| 국면 판별 | MA20/60/120 3중 정렬 → UPTREND / DOWNTREND 판별 |
| 진입 트리거 | MA20이 MA60을 상향 돌파(골든크로스) → UPTREND 내 매수 실행 |
| 청산 트리거 | MA20이 MA60을 하향 돌파(데드크로스) → 2차 부분 매도 |

### 적극투자형 전략 (추가)

| 용도 | 내용 |
|------|------|
| 빠른 진입 트리거 | MA10이 MA20을 상향 돌파 → UPTREND 확인 후 더 빠른 매수 실행 |

> MA10은 국면 판별에는 사용하지 않는다. 국면 판별에 넣으면 4중 정렬 조건이 돼 오히려 신호가 줄어들기 때문이다.
