# RSI · 상대강도지수 (Relative Strength Index)

> 분류: 모멘텀 지표 | 관련 전략: [[투자성향/위험중립형_전략]]

---

## 개념

최근 N일간 상승폭과 하락폭의 비율로 **과매수 / 과매도** 상태를 측정한다.
0~100 범위로 표현되며, 극단값에서 반전 가능성을 시사한다.

```
RSI 70 이상 → 과매수 (너무 많이 오름 → 조정 가능성)
RSI 30 이하 → 과매도 (너무 많이 내림 → 반등 가능성)
```

---

## 계산 방법

```python
delta    = close.diff()
gain     = delta.clip(lower=0)             # 상승분만
loss     = -delta.clip(upper=0)            # 하락분만
avg_gain = gain.ewm(com=window-1).mean()
avg_loss = loss.ewm(com=window-1).mean()
rs       = avg_gain / avg_loss
rsi      = 100 - (100 / (1 + rs))
```

---

## 매매 신호

| 신호 | 조건 | 의미 |
|------|------|------|
| 매수 | RSI가 30 아래에서 **위로** 올라올 때 | 과매도 탈출 → 반등 기대 |
| 매도 | RSI가 70 위에서 **아래로** 내려올 때 | 과매수 탈출 → 조정 기대 |

```python
entries = (rsi > oversold)  & (rsi.shift(1) <= oversold)   # 30 상향 돌파
exits   = (rsi < overbought) & (rsi.shift(1) >= overbought) # 70 하향 돌파
```

---

## 장점 / 단점

| | 내용 |
|---|------|
| **장점** | 과매수/과매도 구간이 명확해 기준 설정이 쉬움 |
| **장점** | 가격보다 먼저 반전 신호를 주는 경우가 많음 |
| **장점** | 볼린저 밴드와 함께 쓰면 횡보장 반전 신호 신뢰도 향상 |
| **단점** | 강한 추세장에서 RSI가 70 이상(또는 30 이하)을 장기간 유지 → 오신호 |
| **단점** | 추세 방향 정보가 없음 — 단독 사용 시 추세 역행 위험 |
| **단점** | 기간(14일) 고정 시 시장 상황에 따라 최적값이 다를 수 있음 |

---

## 이 시스템에서의 역할

현재 메인 전략(위험중립형·적극투자형)에서 **직접 사용하지 않는다.**

SIDEWAYS 국면에서 볼린저 밴드가 RSI와 유사한 역할(과매도 반전 포착)을 이미 담당하고 있어,
RSI를 추가하면 신호 중복이 발생한다.

보조 지표로 구현되어 있으며, 향후 SIDEWAYS 전략 고도화 시 볼린저 밴드와 조합해 활용 가능하다.
