# 모멘텀 (Momentum)

> 분류: 모멘텀 지표 | 관련 전략: [[투자성향/위험중립형_전략]]

---

## 개념

N일 전 대비 현재 가격의 **변화율(수익률)**이다.
값이 클수록 그 기간 동안 강하게 상승했음을 의미한다.

```
모멘텀 > 0  → 해당 기간 동안 상승
모멘텀 < 0  → 해당 기간 동안 하락
모멘텀이 클수록 → 상승 속도가 빠름 (강한 모멘텀)
```

---

## 계산 방법

```python
momentum = close / close.shift(window) - 1   # N일 수익률
```

## 국면별 모멘텀 윈도우

시장 국면에 따라 적절한 모멘텀 측정 기간이 다르다.

| 국면 | 윈도우 | 이유 |
|------|--------|------|
| UPTREND | 126일 (6개월) | 추세가 길게 이어지므로 장기 모멘텀 신뢰도 높음 |
| TRANSITION | 63일 (3개월) | 방향 불확실, 중기 모멘텀으로 중간값 사용 |
| SIDEWAYS | 21일 (1개월) | 단기 등락 반복, 빠른 반응 필요 |
| DOWNTREND | 사용 안 함 | 전량 청산 국면 |

## 장점 / 단점

| | 내용 |
|---|------|
| **장점** | 계산이 단순하고 직관적 |
| **장점** | 종목 간 상대 비교에 유용 — 어떤 종목이 더 강한지 파악 가능 |
| **장점** | 국면별 윈도우를 다르게 설정해 시장 상황에 유연하게 대응 가능 |
| **단점** | 측정 기간 선택에 따라 결과가 크게 달라짐 |
| **단점** | 단기 모멘텀(21일)은 노이즈에 민감 → SIDEWAYS에서 잦은 신호 발생 |
| **단점** | 절대값이 아닌 상대 비교 지표 — 단독으로 진입 여부 판단 불가 |

## 이 시스템에서의 역할

**10번 멀티종목 포트폴리오에서 종목 비중 배분에 사용한다.**

단순 1/N 균등 배분 대신, 모멘텀 강도에 비례해 비중을 배분한다.

```
최종 비중 = 신호 크기 × (종목 모멘텀 / 유효 종목 모멘텀 합)
```

- 음수 모멘텀 종목 → 신호가 발생해도 진입 제외
- 모멘텀이 강한 종목에 자본이 자동으로 집중됨
