# OBV · 거래량 지표 (On-Balance Volume)

> 분류: 거래량 지표 | 관련 전략: [[투자성향/위험중립형_전략]]

---

## 개념

가격이 오른 날은 거래량을 더하고, 내린 날은 빼는 방식으로 **누적 거래량 흐름**을 추적한다.
가격보다 먼저 세력의 매집·분산을 감지하는 **선행 지표**로 활용된다.

```
가격 상승일: OBV += 거래량   (매수 세력 우세)
가격 하락일: OBV -= 거래량   (매도 세력 우세)
가격 보합일: OBV 유지
```

OBV의 절대값은 의미 없다. **추세 방향**만 본다.

---

## 계산 방법

```python
direction = close.diff().apply(lambda x: 1 if x > 0 else (-1 if x < 0 else 0))
obv       = (direction * volume).cumsum()
```

---

## 활용 방법

### 1. OBV 이동평균 돌파 (단독 전략)

```python
obv_ma  = obv.rolling(window).mean()
entries = (obv > obv_ma) & (obv.shift(1) <= obv_ma)   # 매집 시작
exits   = (obv < obv_ma) & (obv.shift(1) >= obv_ma)   # 분산 시작
```

### 2. 다이버전스 (가격 vs OBV 불일치)

| 신호 | 조건 | 의미 |
|------|------|------|
| 강세 다이버전스 | 가격 하락 + OBV 상승 | 세력 매집 → 반등 예고 |
| 약세 다이버전스 | 가격 상승 + OBV 하락 | 거래량 뒷받침 없음 → 하락 예고 |

## 장점 / 단점

| | 내용 |
|---|------|
| **장점** | 가격보다 먼저 세력 움직임을 감지하는 선행 지표 |
| **장점** | 가격 상승이 거래량으로 뒷받침되는지 확인 가능 |
| **장점** | 다이버전스로 추세 전환을 조기에 포착 가능 |
| **단점** | OBV 절대값은 의미 없음 — 추세 방향만 볼 것 |
| **단점** | 다이버전스 판별이 주관적 — 기간 설정에 따라 결과 달라짐 |
| **단점** | 거래량 데이터 품질에 민감 |

---

## 이 시스템에서의 역할

현재 메인 전략(위험중립형·적극투자형)에서 **직접 사용하지 않는다.**

MA + ADX 조합이 추세 확인을 이미 담당하고 있으며,
OBV를 추가하면 신호 복잡도만 증가한다.

보조 지표로 구현되어 있으며, 향후 **진입 신호의 거래량 확인 필터**로 활용 가능하다.
예: 골든크로스 발생 + OBV 상승 동반 시에만 매수 실행.
