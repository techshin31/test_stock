> 분류: 변동성 지표 | 관련 전략: [[투자성향/위험중립형_전략]]

## 개념

하루 동안 가격이 얼마나 움직였는지를 측정하는 **변동성 지표**다.
방향(상승/하락)은 알려주지 않고, 움직임의 **크기**만 측정한다.

```
ATR이 크다 → 가격 변동폭이 넓다 (노이즈 많음)
ATR이 작다 → 가격 변동폭이 좁다 (조용한 장)
```

---

## 계산 방법

True Range는 세 값 중 최댓값이다.

```
TR = max(
    고가 - 저가,              ← 당일 고저 범위
    |고가 - 전일 종가|,        ← 갭 상승 포함
    |저가 - 전일 종가|,        ← 갭 하락 포함
)
```

ATR은 TR의 Wilder's smoothing (지수이동평균 변형) 평균이다.

```python
def calc_atr(high, low, close, period=14):
    prev_close = close.shift(1)
    tr = pd.concat([
        high - low,
        (high - prev_close).abs(),
        (low  - prev_close).abs(),
    ], axis=1).max(axis=1)
    return tr.ewm(com=period - 1, min_periods=period).mean()
```

기본 기간은 14일 (Wilder 권장값).

## Stop-Loss에 ATR을 쓰는 이유

**고정 비율 stop-loss(-7% 등)의 문제:**
- 변동성이 높은 종목: ATR이 5~6%이므로 -7%는 노이즈 범위 안에 들어감 → 불필요한 청산
- 변동성이 낮은 종목: ATR이 1~2%이므로 -7%는 너무 느슨함 → 실제 손실 구간에서 미발동

**ATR 기반 stop-loss의 장점:**
종목과 시기에 따라 노이즈 범위가 다르므로, ATR을 기준으로 삼으면 **노이즈는 허용하고 실제 이탈만 차단**할 수 있다.

```
stop-loss 발동 조건:
  일일 낙폭 < -(전일 ATR / 전일 종가 × multiplier)

예시 (ATR=2%, multiplier=2.0):
  -4% 이상 하락 시 발동 → 노이즈(±2%) 초과 이탈로 판단
```

---

## 이 시스템에서의 역할

**위험중립형 전략의 ATR stop-loss**

| 파라미터 | 기본값 | 의미 |
|----------|--------|------|
| `atr_period` | 14 | ATR 계산 기간 |
| `atr_multiplier` | 2.0 | 낙폭 허용 배수 |

```
발동 조건: 당일 종가 낙폭 < -(전일 ATR / 전일 종가 × 2.0)
결과:      size = 0.0 (즉시 전량 청산) — 최우선 청산 신호
```

**왜 multiplier=2.0인가?**
Wilder의 원 권장값이 2~3이며, 2.0이면 일반적인 일간 노이즈(±ATR)를 2배 넘어설 때만 발동한다.
1.0이면 과도한 청산, 3.0이면 너무 느린 반응이 된다.

**청산 우선순위에서의 위치**
ATR stop-loss는 size_series에서 가장 마지막에 할당되어 모든 앞선 신호를 덮어쓴다.

```
DOWNTREND = 0.0  ← 추세 기반 청산
ATR stop  = 0.0  ← 급락 즉시 청산 (최우선 — DOWNTREND를 덮어씀)
```

## ATR 값 vs ATR 파라미터

ATR에는 두 가지 개념이 섞여 있다.

| 구분 | 설명 | 특성 |
|------|------|------|
| **ATR 값** | 매일 계산되는 실제 변동폭 수치 | 시장 상황에 따라 **매일 변함** |
| **ATR 파라미터** | `atr_period=14`, `atr_multiplier=2.0` | **고정값** — Walk-Forward 최적화 안 함 |

**ATR 값이 매일 바뀐다는 것이 핵심이다.**

```
예시: 삼성전자

조용한 장 (2023년 초):
  ATR = 1,200원, 종가 = 60,000원
  ATR% = 2.0%
  stop-loss 기준 = -(2.0% × 2.0) = -4.0%

급등락 장 (2024년 하반기):
  ATR = 3,000원, 종가 = 60,000원
  ATR% = 5.0%
  stop-loss 기준 = -(5.0% × 2.0) = -10.0%
```

multiplier(2.0)는 고정이지만, **ATR 값이 커지면 stop 기준도 자동으로 넓어진다.**
파라미터를 바꾸지 않아도 시장 변동성에 맞게 스스로 조정되는 것이다.

**왜 Walk-Forward 최적화가 필요 없는가?**

Walk-Forward는 학습 구간의 데이터로 파라미터를 탐색한다.
Stop-loss는 1년 학습 구간에서 0~2번 발동하는 희소 이벤트다.
샘플이 너무 적어 "multiplier=1.5가 좋다" vs "2.0이 좋다"를 구분할 수 없다.

반면 ATR은 변동성 자체가 기준을 조정하므로 최적화 없이도 시장에 적응한다.

## 장점 / 단점

| | 내용 |
|---|------|
| **장점** | ATR 값이 매일 변해 시장 변동성에 자동 적응 — 파라미터 최적화 없이도 작동 |
| **장점** | 고정 비율 대비 오발동(변동성 높은 종목)·미발동(변동성 낮은 종목) 문제 없음 |
| **단점** | 갭 하락(장 시작 전 급락)에는 발동 자체가 다음날로 밀릴 수 있음 |
| **단점** | multiplier 값에 결과가 민감 — 검증 없이 낮추면 과도한 청산 발생 |
