# 볼린저 밴드 (Bollinger Bands)

> 분류: 변동성 지표 | 관련 전략: [[투자성향/위험중립형_전략]] · [[투자성향/적극투자형_전략]]

---

## 개념

중간선(MA20)을 기준으로 **표준편차(σ) × 2배** 만큼 위아래로 밴드를 그린다.
가격이 정상 범위를 벗어났을 때 **평균으로 되돌아오는 성질(평균 회귀)**을 이용한다.

```
상단 밴드 = MA20 + 2σ   → 가격이 여기를 넘으면 과매수
중간선   = MA20         → 기준선
하단 밴드 = MA20 - 2σ   → 가격이 여기를 밑돌면 과매도
```

변동성이 커지면 밴드가 넓어지고, 작아지면 밴드가 좁아진다.

---

## 계산 방법

```python
mid   = close.rolling(window).mean()       # 중간선 (MA20)
std   = close.rolling(window).std()        # 표준편차
upper = mid + num_std * std                # 상단 밴드
lower = mid - num_std * std                # 하단 밴드
```

---

## 매매 신호

| 신호 | 조건 | 의미 |
|------|------|------|
| 매수 | 가격이 하단 밴드를 **아래서 위로** 돌파 | 과매도 → 반등 기대 |
| 매도 | 가격이 상단 밴드를 **위에서 아래로** 돌파 | 과매수 → 조정 기대 |

```python
entries = (close > lower) & (close.shift(1) <= lower.shift(1))
exits   = (close < upper) & (close.shift(1) >= upper.shift(1))
```

## MA(골든크로스)와의 비교

| 항목 | 골든크로스 (MA 기반) | 볼린저 밴드 |
|------|---------------------|------------|
| 전략 유형 | 추세 추종 | 평균 회귀 |
| 적합 시장 | 강한 추세장 | 횡보·박스권 |
| 추세장 성과 | 큰 수익 가능 | 역추세 진입 위험 |
| 횡보장 성과 | 휩소 반복 손실 | 반복 매매로 수익 |

---

## 장점 / 단점

| | 내용 |
|---|------|
| **장점** | 횡보장에서 명확한 진입(하단) / 청산(상단) 기준 제공 |
| **장점** | 변동성이 커질수록 밴드가 넓어져 신호가 자동 조정됨 |
| **장점** | 직관적 시각화 → 전략 이해와 디버깅이 쉬움 |
| **단점** | 강한 추세 발생 시 밴드를 계속 돌파 → 오신호 반복 |
| **단점** | 변동성이 극도로 낮은 장세에서 밴드 수축 → 신호 거의 없음 |
| **단점** | 하단 터치가 반등이 아닌 추가 하락의 시작일 수 있음 |

## 이 시스템에서의 역할

**SIDEWAYS 국면에서만 사용한다.**

추세장(UPTREND/DOWNTREND)에서 볼린저 밴드를 쓰면 오신호가 많다.
ADX < 20으로 횡보를 확인한 후에만 볼린저 밴드를 활성화해 이 문제를 해결한다.

```
SIDEWAYS 확인 (ADX < 20)
  → 하단 밴드 터치 → 매수 (목표 비중 50%)
  → 상단 밴드 터치 → 전량 청산
```

UPTREND에서는 [[TA지표/추세/MA_이동평균|골든크로스]]가 역할을 대신한다.
