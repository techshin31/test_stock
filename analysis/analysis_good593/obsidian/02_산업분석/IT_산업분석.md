## 분석 내용

### 1. 시가총액 시계열 (월별)
- IT 섹터 전체 시가총액을 월별로 집계
- 소분류별 시총 비중 변화 추이 파악

### 2. 소분류별 시가총액 비중 (최신: 2026-03-27)
- 각 IT 소분류가 전체 IT 섹터에서 차지하는 비중 분석

### 3. 글로벌 자산과의 상관관계

> 월별 시총 vs 글로벌 자산 지표 상관계수

| 자산 지표 | 상관 수준 |
|-----------|-----------|
| 반도체(SOX) | **높음 (|r| > 0.7)** |
| 구리(HG=F) | **높음 (|r| > 0.7)** |
| 금(GC=F) | **높음 (|r| > 0.7)** |
| 석유(WTI) | 중간 |
| 해운운임(BDI) | 낮음 |
| 미국 10년 국채 | 중간 |
| 미국 CPI | 낮음 |

### 4. 주요 인사이트

- **반도체(SOX)가 가장 강한 상관관계**: 한국 IT 섹터(특히 반도체 기업 비중이 높음)는 글로벌 반도체 지수와 동조화가 강함
- **구리·금도 높은 상관**: 글로벌 경기 확장 국면에서 IT 섹터 시총도 함께 상승
- **CPI 영향 미미**: 인플레이션 환경보다 기술 사이클이 IT 섹터를 주도

### 5. 정규화 비교 (시작=100 기준)
- IT 시총과 반도체(SOX), 구리, 금이 비슷한 상승 궤적을 그림
- 특히 2023~2024년 AI 반도체 붐 시기에 동반 급등
