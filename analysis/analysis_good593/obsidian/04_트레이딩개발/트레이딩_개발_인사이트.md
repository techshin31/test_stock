## 인사이트 #1: Global Expansion Score

### 배경

IT·경기소비재·산업재 3개 섹터 모두 **구리·금·SOX와 동시에 |r| > 0.7** 상관관계를 보임.
이 3개 자산을 통합한 스코어가 섹터 비중 조절의 1차 필터로 활용 가능.

### 계산식

```python
# asset_combined_close.csv 기반
def global_expansion_score(df):
    sox_norm   = (df['반도체(SOX)']  - df['반도체(SOX)'].mean())  / df['반도체(SOX)'].std()
    copper_norm = (df['구리(HG=F)']  - df['구리(HG=F)'].mean())   / df['구리(HG=F)'].std()
    gold_norm  = (df['금(GC=F)']    - df['금(GC=F)'].mean())     / df['금(GC=F)'].std()
    return (sox_norm + copper_norm + gold_norm) / 3
```

### 현재 상태

- SOX +75%, 구리 +27%, 금 +86% → 3개 모두 5년 평균 대비 강세
- **Global Expansion Score 최고조 → 리스크 온 상태**

### 개발 태스크

- [ ] `asset_combined_close.csv` 로 월별 Global Expansion Score 계산
- [ ] 스코어 임계값(예: > 1.0 / 0~1.0 / < 0) 기준 섹터 비중 백테스트
- [ ] 스코어 추세 반전 감지 알림 모듈

## 인사이트 #2: 섹터별 분리 모델

### 섹터 모델 분류표

| 섹터 | WICS | 모델 타입 | 주요 피처 | 제외 피처 |
|------|------|-----------|----------|----------|
| IT | G45 | 글로벌 연동 모델 | SOX, 구리, 금 | CPI |
| 경기소비재 | G25 | 글로벌 연동 모델 | SOX, 구리, 금 | CPI |
| 산업재 | G20 | 글로벌+인플레 모델 | SOX, 구리, 금, **CPI** | - |
| **소재** | **G15** | **독립 도메인 모델** | 국내 배터리 수요, 중국 철강, 정책 | 글로벌 자산 전체 |
| 필수소비재 | G30 | 방어주 모델 | 금, 구리 | SOX, 국채 |

### 소재(G15) 독립 모델 설계 필요

- 글로벌 자산 중 |r| > 0.7 지표 **없음** → 별도 피처 파이프라인 필수
- 국내 고유 테마: 2차전지 수요, 정부 친환경 정책, 중국 저가 소재 공급
- 추가 수집 필요 데이터: POSCO 수주잔고, 국내 배터리 출하량, LG화학 실적 가이던스

### 개발 태스크

- [ ] 섹터별 피처셋 분리 파이프라인 구현
- [ ] 소재(G15) 전용 국내 지표 데이터 수집 모듈 추가
- [ ] 산업재(G20): CPI 월별 피처 연동 (현재 BDI와 낮은 상관 → BDI 제외)

## 인사이트 #3: 섹터 로테이션 룰

### 로테이션 조건표

```
[리스크 온 - 확장기]
IF Global_Expansion_Score > 1.0:
    IT(G45) 비중 ↑
    경기소비재(G25) 비중 ↑

[인플레이션 구간]
IF CPI YoY > 4% AND 구리 강세:
    산업재(G20) 추가 비중 ↑
    (건설·조선·방산 수주 증가 효과)

[리스크 오프 - 방어 전환]
IF 금 강세 AND SOX 약세:
    필수소비재(G30) 비중 ↑
    (국채 금리 민감도 낮음 → 고금리 헷지 유리)

[소재 독립 신호]
IF 국내 2차전지 정책 이슈 OR 중국 철강 수요 변화:
    소재(G15) 독립 신호 감시 모드 진입
```

### 필수소비재(G30) 헷지 특성

- 미국 10년 국채 금리와 상관 **낮음** → 현재 4.44% 고금리 환경에서 헷지 수단으로 적합
- 금·구리 상승 구간에서 완만하게 동반 상승 → 방어적 상승 가능

### 개발 태스크

- [ ] 로테이션 룰 기반 월별 섹터 비중 자동 계산 모듈
- [ ] 백테스트: 2021~2026 구간 섹터 로테이션 수익률 vs 코스피 벤치마크
- [ ] 로테이션 신호 변화 시 알림 발송

|------|-----|------|

| 배당수익률 | ~9~10조/년 | 안정적 배당 유지 |

### 진입/청산 조건

```
[매수 신호]
SOX 상승 추세 유지 (현재 충족)
AND PER < 15배 (현재 12.6배로 충족)
AND HBM 공급 지연 이슈 해소 신호 (SK하이닉스와의 격차 모니터링)

[비중 축소 신호]
SOX가 5년 평균 대비 100% 이상 괴리 (현재 +75%, 아직 여유 있음)
OR WTI 급등 → CPI 반등 → 금리 인하 기대 후퇴
OR HBM 경쟁 심화 (엔비디아 공급사 점유율 변화)
```

### 주요 리스크 모니터링 항목

| HBM 경쟁 | SK하이닉스 HBM 점유율 | 60% 초과 지속 시 리스크 ↑ |
| 지정학 | 대중 수출 규제 뉴스 | 이벤트 기반 감지 |
| 스마트폰 | 글로벌 출하량 YoY | -5% 이하 시 MX 사업 우려 |

## 인사이트 #5: WTI vs BDI 괴리 리스크

### 현재 상황

```
BDI -23% (공급망 정상화) → 물류비 안정 → 수입 원가 유리
WTI +30% (지정학 리스크) → 에너지 비용 압박 → CPI 재상승 잠재 리스크
```

### 리스크 시나리오

- WTI가 100달러 이상 유지 → CPI 2% 목표 달성 지연 → Fed 금리 인하 후퇴
- 고금리 장기화 → PER 멀티플 억제 → IT·경기소비재 과열 조정 가능성
- 현재 SOX +75%, 금 +86% 동시 과열 → **포지션 사이즈 보수적 관리 필요**

### 개발 태스크

- [ ] WTI + CPI 교차 신호 모듈: WTI 월평균 > 95달러 지속 시 CPI 반등 경고
- [ ] 금리 인하 기대 지수 연동 (Fed Fund Futures 데이터 수집)
- [ ] 포지션 사이즈 자동 조절: Global Expansion Score 과열 구간 진입 시 레버리지 하향
